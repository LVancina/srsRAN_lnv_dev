This class adapts srs\+RAN scheduler interface to operate with srs\+RAN MAC. The configuration completion notification handling (e.\+g. ue creation complete) is deferred for later processing rather than being processed inline. We defer the processing because we do not want it to take place while the scheduler is still processing the slot\+\_\+indication, given that latter has higher priority.
