\doxysubsection{Detailed Description}
Object used to connect the DU and CU-\/\+UP F1-\/U bearers On the co-\/located case this is done by connecting both entities directly. 

Note that CU and DU bearer creation and removal can be performed from different threads and are therefore protected by a common mutex.
