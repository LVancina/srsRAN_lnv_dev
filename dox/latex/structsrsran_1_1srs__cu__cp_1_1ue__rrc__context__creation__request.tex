\doxysubsection{Detailed Description}
Request made by the F1\+AP-\/\+CU to create a RRC context for an existing UE context in the CU-\/\+CP. 

This request should be made once the C-\/\+RNTI and cell of the UE is known. That generally corresponds to the moment a Initial UL RRC Message or a F1\+AP UE Context Setup Response are received.
