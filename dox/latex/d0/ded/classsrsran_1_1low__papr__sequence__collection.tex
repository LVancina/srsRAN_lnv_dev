\doxysubsection{Detailed Description}
Describes a low PAPR sequence collection as per TS38.\+211 Section 5.\+2.\+2. 

This class describes a collection of $r^{(\alpha, \delta)}_{u,v}(n)$ sequences. The collection of sequences is generated at construction time. The constructor is not suitable for real-\/time operation while the sequence retrieval through method \doxylink{classsrsran_1_1low__papr__sequence__collection_acec34842f3a9824730c9cd5e87d6f998}{get()} is suitable for real-\/time execution.
