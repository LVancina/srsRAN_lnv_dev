\doxysubsection{Detailed Description}
This interface represents the data entry point of the transmitting side of a RLC entity. The upper-\/layer will use this call to pass RLC SDUs into the TX entity. The upper-\/layer will also inform RLC of SDUs (e.\+g. PDCP PDUs) to be discarded.
