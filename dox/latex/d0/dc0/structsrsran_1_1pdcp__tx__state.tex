This state variable indicates the lower edge of the TX window, i.\+e. the COUNT value of the oldest PDCP SDU in the TX window for which the discard timer is expected to expire soonest. If TX\+\_\+\+NEXT\+\_\+\+ACK == TX\+\_\+\+NEXT, it means we are not currently having any discard timers running. For UM bearers this value is the same as TX\+\_\+\+TRANS.
