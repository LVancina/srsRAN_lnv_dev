\doxysubsection{Detailed Description}
PRACH buffer interface. 

Provides access to frequency-\/domain PRACH sequences.

A buffer for storing long PRACH sequences ( $L_{RA}=839$), for a maximum of 4 OFDM symbols, can be created with the factory function \doxylink{namespacesrsran_ac2e894cb68f8d4b6e878695cf6db27eb}{create\+\_\+prach\+\_\+buffer\+\_\+long()}. A buffer for storing short PRACH sequences ( $L_{RA}=129$), for a maximum of 12 OFDM symbols, can be created with the factory function \doxylink{namespacesrsran_a771326243d595c8028b9e233d69a51e1}{create\+\_\+prach\+\_\+buffer\+\_\+short()}.
