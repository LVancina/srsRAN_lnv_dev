\subsubsection*{template$<$typename Ty, typename Name, typename Units$>$\newline
struct srslog\+::metric$<$ Ty, Name, Units $>$}
A metric is the most basic object that composes a context. It is generally used to represent any kind of state of a program. It stores a value of type T associated with a name and the units.
