\doxysection{srsran\+::byte\+\_\+buffer Class Reference}
\hypertarget{classsrsran_1_1byte__buffer}{}\label{classsrsran_1_1byte__buffer}\index{srsran::byte\_buffer@{srsran::byte\_buffer}}


Byte sequence, which represents its data in memory via an intrusive linked list of memory chunks. This container is not contiguous in memory. Default copy ctor and assignment is disabled in this container. The user should instead std\+::move to transfer ownership, .\doxylink{classsrsran_1_1byte__buffer_a1f08be439b9c2c174b4b2827c18feda6}{copy()} for shallow copies with shared ownership and .\doxylink{classsrsran_1_1byte__buffer_aa79a5e270dbb5dd01b83c034853b9fb3}{deep\+\_\+copy()} for byte-\/wise copies.

\doxysubsection{Detailed Description}
Byte sequence, which represents its data in memory via an intrusive linked list of memory chunks. This container is not contiguous in memory. Default copy ctor and assignment is disabled in this container. The user should instead std\+::move to transfer ownership, .\doxylink{classsrsran_1_1byte__buffer_a1f08be439b9c2c174b4b2827c18feda6}{copy()} for shallow copies with shared ownership and .\doxylink{classsrsran_1_1byte__buffer_aa79a5e270dbb5dd01b83c034853b9fb3}{deep\+\_\+copy()} for byte-\/wise copies.
