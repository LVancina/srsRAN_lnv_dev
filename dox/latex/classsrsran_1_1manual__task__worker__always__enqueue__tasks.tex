\doxysubsection{Detailed Description}
Task worker that implements the executor interface and requires manual calls to run pending deferred tasks. Useful for unit testing. 

\begin{DoxyNote}{Note}
This implementation always queues the tasks, either by calling execute or defer. 
\end{DoxyNote}
