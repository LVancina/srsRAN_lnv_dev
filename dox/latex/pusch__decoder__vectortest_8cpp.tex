\doxysubsection{Detailed Description}
Unit test for PUSCH decoder. 

Vector-\/based unit test. Each test vector contains the UL-\/\+SCH transport block and the LLRs corresponding to multiple retransmissions with different redundancy versions. For each retransmission, the test combines the received codeword with the LLRs of previous transmissions (if any) and decodes the resulting sequence of LLRs. The test vectors have been generated with very little noise, so that the decoding should be successful after each retransmission with only few LDPC iterations.
