Returns the compact representation of the lifted node at position $(m,n)$. 

If the base graph has an edge between check node {\ttfamily m} and variable node {\ttfamily n}, the returned value is the shift applied to the lifted node. Conversely, if there is no edge, then the returned value is srsran\+::ldpc\+::\+NO\+\_\+\+EDGE.
