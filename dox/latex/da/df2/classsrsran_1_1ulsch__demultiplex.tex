\doxysubsection{Detailed Description}
User interface of the Uplink Shared Channel (UL-\/\+SCH) demultiplexer. 

Note that, for calculating the CSI Part 2 report size, it is necessary to decode the CSI Part 1. Consequently, \doxylink{classsrsran_1_1ulsch__demultiplex_af937e011d9c1196511706e6c5aef7544}{ulsch\+\_\+demultiplex\+::set\+\_\+csi\+\_\+part2} must be called with the size information obtained from CSI Part 1 before the CSI Part 2 report can be demultiplexed.

All fields can be demultiplexed simultaneously if CSI Part 2 is not present (see \doxylink{classsrsran_1_1ulsch__demultiplex_a4f27384b287c26d50aefdda91c897726}{ulsch\+\_\+demultiplex\+::demultiplex}).
