An index determining the position of the first bit of TPC command (applicable to the PUCCH SCell) inside the DCI format 2-\/2 payload. Values \{1..15\}. The field is mandatory present if the PUCCH-\/\+TPC-\/\+Command\+Config is provided in the PDCCH-\/\+Config for the PUCCH-\/\+SCell. The field is optionally present, need R, if the UE is configured with a PUCCH SCell in this cell group and if the PUCCH-\/\+TPC-\/\+Command\+Config is provided in the PDCCH-\/\+Config for the Sp\+Cell.
