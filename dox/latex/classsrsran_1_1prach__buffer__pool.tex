\doxysubsection{Detailed Description}
Pool of PRACH buffers. 

As the pool owns the PRACH buffers that it stores, the lifespan of the buffers is determined by lifespan of the pool.

\begin{DoxyNote}{Note}
The pool must ensure that, when assigning a PRACH buffer upon request, the buffer is not currently being used. 
\end{DoxyNote}
