\doxysubsection{Detailed Description}
Describes an LDPC segment or codeblock. 

The segment description consists of a set of parameters contained in metadata attribute and a data storage for the segment.

If the object is created with the default constructor, the {\ttfamily \doxylink{classsrsran_1_1described__segment_a9d94f1b666ce172f87abc5bd001d1307}{get\+\_\+data()}} methods trigger an assertion.
