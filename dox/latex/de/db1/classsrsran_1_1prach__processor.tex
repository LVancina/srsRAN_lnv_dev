\doxysubsection{Detailed Description}
PRACH processor main interface. 

Provides a single access point to all PRACH-\/related interfaces, that is \doxylink{classsrsran_1_1prach__processor__notifier}{prach\+\_\+processor\+\_\+notifier}, \doxylink{classsrsran_1_1prach__processor__request__handler}{prach\+\_\+processor\+\_\+request\+\_\+handler} and \doxylink{classsrsran_1_1prach__processor__baseband}{prach\+\_\+processor\+\_\+baseband}.

\begin{DoxyNote}{Note}
All implementations of this interface must trigger an assertion error if the PRACH processor is not connected to a notifier before calling \doxylink{classsrsran_1_1prach__processor_aa17198bdd7554b7c5c039e39c57cd5c1}{get\+\_\+request\+\_\+handler()} or \doxylink{classsrsran_1_1prach__processor_a284eea67f6f4aa8d3017dd52ed9e8a7b}{get\+\_\+baseband()}. 
\end{DoxyNote}
