\doxysubsection{Detailed Description}
Log-\/likelihood ratio type declaration. 

Type used for representing log-\/likelihood ratios. Values are stored in an {\ttfamily int8\+\_\+t} variable, but the sum operator is overloaded with a saturated sum, meaning that the result is clipped if it (absolutely) exceeds {\ttfamily LLR\+\_\+\+MAX}.
