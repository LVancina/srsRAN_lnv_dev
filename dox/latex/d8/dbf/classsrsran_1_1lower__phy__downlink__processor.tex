\doxysubsection{Detailed Description}
Downlink processor main interface. 

Provides a single access point to all downlink-\/related interfaces, that is \doxylink{classsrsran_1_1downlink__processor__notifier}{downlink\+\_\+processor\+\_\+notifier}, \doxylink{namespacesrsran_a58e7c9f2c51a9ad6a0899b46d5315723}{downlink\+\_\+processor\+\_\+request\+\_\+handler} and \doxylink{classsrsran_1_1downlink__processor__baseband}{downlink\+\_\+processor\+\_\+baseband}.

\begin{DoxyNote}{Note}
All implementations of this interface must trigger an assertion error if the downlink processor is not connected to a notifier before calling \doxylink{classsrsran_1_1lower__phy__downlink__processor_a210ee414de55d8d5b66fb7881e0d4c0f}{get\+\_\+downlink\+\_\+request\+\_\+handler()} or \doxylink{classsrsran_1_1lower__phy__downlink__processor_ac7838dc570fbbd8d782eef8304c59795}{get\+\_\+baseband()}. 
\end{DoxyNote}
