\doxysubsection{Detailed Description}
Implementation of the uplink request processor interface. 

This implementation processes uplink data requests according to the following steps.
\begin{DoxyEnumerate}
\item It resolves all the dependencies.
\item It calls a method from the internal \doxylink{classsrsran_1_1upper__phy__rx__symbol__request__notifier}{upper\+\_\+phy\+\_\+rx\+\_\+symbol\+\_\+request\+\_\+notifier} object. This will produce and notify an event that requests the capture of slot symbol data. 
\end{DoxyEnumerate}
