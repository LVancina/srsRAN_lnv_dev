\doxysubsection{Detailed Description}
Downlink processor class that groups and process all the downlink channels within a slot. 

The downlink processor process all the given downlink PDUs and sends the configured resource grid through a gateway when every PDU has finished processing and the \doxylink{classsrsran_1_1downlink__processor_af279e448ba47bb0008b4786a3983c5d5}{finish\+\_\+processing\+\_\+pdus()} has been called. Prior to start processing PDUs, \doxylink{classsrsran_1_1downlink__processor_af578458d6d55670d62a9bf920b5f0bf4}{configure\+\_\+resource\+\_\+grid()} must be called in order to configure the resource grid and the context for the downlink processor.

\begin{DoxyNote}{Note}
Undefined behavior can be caused by\+:
\begin{DoxyItemize}
\item Calling any process method prior to \doxylink{classsrsran_1_1downlink__processor_af578458d6d55670d62a9bf920b5f0bf4}{configure\+\_\+resource\+\_\+grid()}.
\item Calling \doxylink{classsrsran_1_1downlink__processor_af279e448ba47bb0008b4786a3983c5d5}{finish\+\_\+processing\+\_\+pdus()} without calling \doxylink{classsrsran_1_1downlink__processor_af578458d6d55670d62a9bf920b5f0bf4}{configure\+\_\+resource\+\_\+grid()} in a slot context. 
\end{DoxyItemize}
\end{DoxyNote}
