\doxysubsection{Detailed Description}
Uplink processor interface. 

The uplink processor is in charge of handling incoming requests to process the physical uplink channels within a certain slot.

Requests are dispatched asynchronously as they get enqueued for execution, and generate an event through the \doxylink{classsrsran_1_1upper__phy__rx__results__notifier}{upper\+\_\+phy\+\_\+rx\+\_\+results\+\_\+notifier} interface upon completion.
