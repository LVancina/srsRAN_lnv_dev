\doxysubsection{Detailed Description}
Describes CORESET parameters. 

Provides the Control Resource Set (CORESET, TS38.\+331 {\ttfamily Control\+Resource\+Set} for more information) related parameters common for up to {\ttfamily MAX\+\_\+\+NOF\+\_\+\+DCI} Downlink Control Information (DCI) transmissions.

For {\ttfamily control\+Resource\+Set\+Zero} (CORESET 0, TS38.\+213 Section 13 for more information) there are some exceptions\+:
\begin{DoxyItemize}
\item {\ttfamily cce\+\_\+to\+\_\+reg\+\_\+mapping\+\_\+type} shall be set to {\ttfamily CORESET0}.
\item {\ttfamily bwp\+\_\+start\+\_\+rb} and {\ttfamily bwp\+\_\+size\+\_\+rb} indicate the CORESET start point and contiguous resource blocks.
\item {\ttfamily shift\+\_\+index} shall be equal to the physical cell identifier \{0...1007\}.
\item {\ttfamily cce\+\_\+to\+\_\+reg\+\_\+mapping\+\_\+type}, {\ttfamily frequency\+\_\+resources}, {\ttfamily reg\+\_\+bundle\+\_\+size}, {\ttfamily interleaver\+\_\+size} are ignored. 
\end{DoxyItemize}
