\doxysubsection{Detailed Description}
Contains the PRACH configuration parameters. 

The parameters are used in TS38.\+211 Section 6.\+3.\+3.\+2 and they are derived from TS38.\+211 Tables 6.\+3.\+3.\+2-\/2, 6.\+3.\+3.\+2-\/3 and 6.\+3.\+3.\+2-\/4.

The PRACH transmission occasions are in the system frames $n_{SFN}$ that satisfy $n_{SFN} \bmod x = y$.
