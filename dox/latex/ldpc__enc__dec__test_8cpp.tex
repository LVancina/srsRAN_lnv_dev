\doxysubsection{Detailed Description}
Unit test for LDPC encoder and decoder. 

For all possible base graphs and lifting sizes, the test extracts from a file a small set of messages and corresponding codeblocks. The messages are fed to the encoder, whose output is compared to the codeblocks. Similarly, the codeblocks are fed to the decoder and the resulting messages are compared to the example ones.
