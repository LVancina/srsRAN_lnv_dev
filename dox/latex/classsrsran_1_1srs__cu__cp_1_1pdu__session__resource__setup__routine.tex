\doxysubsection{Detailed Description}
Handles the setup of PDU session resources from the CU-\/\+CP point of view. 

The routine combines several (sub-\/)procedures involving the CU-\/\+UP and the DU. Depending on the current state of the UE and bearer context it may involve\+:
\begin{DoxyItemize}
\item Initiate or modifiy the CU-\/\+UPs bearer context over E1\+AP
\item Modify the DU\textquotesingle{}s UE context over F1\+AP
\item Modify the CU-\/\+UPs bearer context over E1\+AP (update TEIDs, etc)
\item Modify the UE\textquotesingle{}s configuration over RRC signaling
\end{DoxyItemize}

All procedure are executed sequentially and their outcome and the contained values in the result messages may affect the next procedure\textquotesingle{}s request content. The general paradigm of execution is that when processing the result message of one procedure in handle\+\_\+procedure\+\_\+response() the content for the subsequent request is already pre-\/filled based on the processed result.
