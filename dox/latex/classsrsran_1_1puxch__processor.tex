\doxysubsection{Detailed Description}
PUCCH and PUSCH processor main interface. 

Provides a single access point to all PUCCH-\/ and PUSCH-\/related interfaces, that is \doxylink{classsrsran_1_1puxch__processor__notifier}{puxch\+\_\+processor\+\_\+notifier}, \doxylink{classsrsran_1_1puxch__processor__request__handler}{puxch\+\_\+processor\+\_\+request\+\_\+handler} and \doxylink{classsrsran_1_1puxch__processor__baseband}{puxch\+\_\+processor\+\_\+baseband}.

\begin{DoxyNote}{Note}
All implementations of this interface must trigger an assertion error if the PUx\+CH processor is not connected to a notifier before calling \doxylink{classsrsran_1_1puxch__processor_a2d86fab8de5cbe8e737a7f2184460364}{get\+\_\+request\+\_\+handler()} or \doxylink{classsrsran_1_1puxch__processor_a1a557335a65df4e3d6ba6829d1121c76}{get\+\_\+baseband()}. 
\end{DoxyNote}
