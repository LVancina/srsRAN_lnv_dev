\doxysubsection{Detailed Description}
DCI configuration parameters required to perform the DCI size alignment procedure. 

The size alignment procedure, specified in TS38.\+212 Section 7.\+3.\+1.\+0, reduces the number of different DCI sizes, to simplify the detection of DCI messages by the UE.

\begin{DoxyRemark}{Remarks}
CORESET 0 is assumed to be configured for the cell if \doxylink{structsrsran_1_1dci__size__config_a93a3a92f400bb4f26567a8a30d4b103c}{coreset0\+\_\+bw} is set to any value other than 0. 
\end{DoxyRemark}
