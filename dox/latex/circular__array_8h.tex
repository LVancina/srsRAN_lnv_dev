\doxysubsection{Detailed Description}
Helper class to safely access elements of a std\+::array. 

Protects from out-\/of-\/bounds access by applying modulo of it\textquotesingle{}s length when using the \mbox{[}\mbox{]} operator for element access.

This is useful for circular data structures, like TTIs or SNs.
