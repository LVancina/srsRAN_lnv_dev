\doxysubsection{Detailed Description}
Non-\/standard configurable parameters for PDCP. 

Configurable parameters for the PDCP entity that are not explicitly specified in the RRC specifications. This separation is necessary, as the CU-\/\+CP cannot transmit these parameters to the CU-\/\+UP, so it\textquotesingle{}s necessary for the CU-\/\+UP to store these configurations itself.
