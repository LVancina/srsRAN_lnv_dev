\doxysubsection{Detailed Description}
This interface represents the data entry point of the transmitting side of a F1-\/U entity. The upper-\/layers will use this class to pass PDCP PDUs into the TX entity. The upper-\/layers will use this class to inform lower layers of PDCP PDUs to be discarded.
