\subsubsection*{template$<$\mbox{\hyperlink{namespacesrsran_a58e7c9f2c51a9ad6a0899b46d5315723}{typename}} Data$>$\newline
class srsran\+::manual\+\_\+event$<$ Data $>$}
Awaitable type that can be manually set to one of the states\+: 


\begin{DoxyItemize}
\item "{}unset"{} state -\/ Coroutines that await on an "{}unset"{} \doxylink{classsrsran_1_1manual__event}{manual\+\_\+event} get suspended. They are only resumed when the \doxylink{classsrsran_1_1manual__event}{manual\+\_\+event} transitions to "{}set"{} state via the \doxylink{classsrsran_1_1manual__event_af7395d3c9cd8d17ff5bf0c7b42b94e95}{manual\+\_\+event\+::set} method.
\item "{}set"{} state -\/ Coroutines that await on a "{}set"{} \doxylink{classsrsran_1_1manual__event}{manual\+\_\+event} do not get suspended and get automatically resumed. Once "{}set"{}, the awaiting coroutines have to option to extract the {\ttfamily Data} that was stored in the \doxylink{classsrsran_1_1manual__event}{manual\+\_\+event}. Multiple coroutines can simultaneously await this awaitable when in "{}unset"{} state. All the awaiting coroutines will be resumed once the awaitable is set.
