class srsran\+::detail\+::ring\+\_\+buffer\+\_\+impl$<$ T, Container, Force\+Power2\+Size $>$}
Base common class for definition of ring buffer data structures with the following features\+:
\begin{DoxyItemize}
\item no allocations while pushing/popping new elements. Just an internal index update
\item it provides an api to push/pop elements like a queue
\item it provides an iterator interface to iterate over added elements in the buffer
\item not thread-\/safe.
