\doxysubsubsection{\texorpdfstring{dl\_bytes\_per\_slot}{dl\_bytes\_per\_slot}}
{\footnotesize\ttfamily unsigned bench\+\_\+params\+::dl\+\_\+bytes\+\_\+per\+\_\+slot = 1500}



Set the number of bytes pushed to the DU DL F1-\/U interface per slot. 

Setting this value to 0 will disable DL Tx. If the air interface cannot keep up with the DL F1-\/U PDU rate, the F1-\/U will be throttled, to avoid depleting the buffer pool.
