\doxysubsection{Detailed Description}
PUSCH codeword buffer interface. 

Interfaces the PUSCH demodulator with the UL-\/\+SCH demultiplexer. The PUSCH demodulator notifies processed continuous blocks of data using the method \doxylink{classsrsran_1_1pusch__codeword__buffer_a502fcdeac2afa2fca80b6b70ceaeade3}{on\+\_\+new\+\_\+block} and notifies the end of data using the method \doxylink{classsrsran_1_1pusch__codeword__buffer_a134e2279ce425f05c0cf0c1cd01bfd65}{on\+\_\+end\+\_\+codeword}.
