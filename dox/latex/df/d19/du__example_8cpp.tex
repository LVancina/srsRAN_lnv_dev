\doxysubsection{Detailed Description}
Example application of a distributed unit (DU) transmitting SIB1 over a radio interface. 

This example runs a DU without an F1 connection to a CU. It integrates the DU high and DU low subcomponents and connects them through the FAPI interface, the DU low is then connected to the lower PHY and a real-\/time radio interface. The example transmits SIB1 messages.

The application supports different working profiles, run {\ttfamily  du\+\_\+example -\/h } for usage details.
