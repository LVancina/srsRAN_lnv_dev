\doxysubsection{Detailed Description}
A symbol level bitmap in time domain. It indicates with a bit set to true that the UE shall rate match around the corresponding symbol. This pattern recurs (in time domain) with the configured periodicity\+And\+Pattern. \begin{DoxyRemark}{Remarks}
See TS 38.\+214, clause 5.\+1.\+4.\+1. For ECP the first 12 bits represent the symbols within the slot and the last two bits within the bitstring are ignored by the UE. 
\end{DoxyRemark}
