\doxysubsection{Detailed Description}
Describes a Discrete Fourier Transform (DFT) processor. 

\begin{DoxyRemark}{Remarks}
The DFT processor allocates the input and output buffers for optimization purposes. 

The input buffer is available using \doxylink{classsrsran_1_1dft__processor_aa07d386497c47ca798864f6300937cab}{get\+\_\+input()}. 

The output buffer is available using \doxylink{classsrsran_1_1dft__processor_a628d1ee463ea4107a2809eeb6cdac552}{run()}. 
\end{DoxyRemark}
