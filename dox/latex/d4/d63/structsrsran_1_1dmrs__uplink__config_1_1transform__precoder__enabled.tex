{\footnotesize\ttfamily \mbox{\hyperlink{classbool}{bool}} srsran\+::dmrs\+\_\+uplink\+\_\+config\+::transform\+\_\+precoder\+\_\+enabled\+::is\+\_\+seq\+\_\+grp\+\_\+hopping\+\_\+disabled \{\mbox{\hyperlink{namespacesrsran_a58e7c9f2c51a9ad6a0899b46d5315723}{false}}\}}

For DMRS transmission with transform precoder the NW may configure group hopping by the cell-\/specific parameter group\+Hopping\+Enabled\+Transform\+Precoding in PUSCH-\/\+Config\+Common. In this case, the NW may include this UE specific field to disable group hopping for PUSCH transmission except for Msg3. Override the configuration in PUSCH-\/\+Config\+Common. Mapping to NR RRC\+: If true, the field is present in NR RRC indicating value disabled to UE. If false, the field is absent in NR RRC and the UE uses the same hopping mode as for Msg3.

{\footnotesize\ttfamily \mbox{\hyperlink{classbool}{bool}} srsran\+::dmrs\+\_\+uplink\+\_\+config\+::transform\+\_\+precoder\+\_\+enabled\+::is\+\_\+seq\+\_\+hopping\+\_\+enabled \{\mbox{\hyperlink{namespacesrsran_a58e7c9f2c51a9ad6a0899b46d5315723}{false}}\}}

Determines if sequence hopping is enabled for DMRS transmission with transform precoder for PUSCH transmission other than Msg3 (sequence hopping is always disabled for Msg3). Mapping to NR RRC\+: If true, the field is present in NR RRC indicating value enabled to UE. If false, the field is absent in NR RRC and the UE uses the same hopping mode as for Msg3. Note\+: The network does not configure simultaneous group hopping and sequence hopping.
