\doxysubsection{Detailed Description}
PDSCH encoder, as per TS38.\+212 Section 7.\+2. 

For a PHY Downlink Shared Data CHannel (PDSCH) transport block, it returns the single codeword obtained after CRC attachment, segmentation, codeblock encoding, rate matching and, finally, codeblock concatenation.
