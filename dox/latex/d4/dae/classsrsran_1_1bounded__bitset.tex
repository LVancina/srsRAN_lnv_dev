\doxysubsection{Detailed Description}
\subsubsection*{template$<$size\+\_\+t N, \mbox{\hyperlink{classbool}{bool}} Lowest\+Info\+Bit\+Is\+MSB = false$>$\newline
class srsran\+::bounded\+\_\+bitset$<$ N, Lowest\+Info\+Bit\+Is\+MSB $>$}
Represents a dynamically-\/sized bitset with an upper bound capacity of N bits. 

The \doxylink{classsrsran_1_1bounded__bitset}{bounded\+\_\+bitset} is represented internally via an array of uint64\+\_\+t, with each integer storing a bitmap. This class also offers many standard logic manipulation methods, like any(), operators \&=, \&, \texorpdfstring{$\vert$}{|}=, \texorpdfstring{$\vert$}{|}, etc. and utility methods to convert the bitset into strings or integers.

Depending on the passed {\ttfamily Lowest\+Info\+Bit\+Is\+MSB} template argument, this class can represent the bits of the bitset in different orders. E.\+g.

bounded\+\_\+bitset$<$6, false$>$ a(5); // Bitset of 5 bits. LSB is Lowest Information Bit (bit 0). a.\+set(0); assert(a.\+to\+\_\+uint64() == 0b1); bounded\+\_\+bitset$<$6, true$>$ b(5); // Bitset of 5 bits. MSB is Lowest Information Bit (bit 0). b.\+set(1); assert(a.\+to\+\_\+uint64() == 0b1000);

The {\ttfamily Lowest\+Info\+Bit\+Is\+MSB} template argument also affects the default string representation of the bitset. E.\+g. fmt\+::print("{}\{\+:b\}"{}, a); // prints "{}00001"{}. fmt\+::print("{}\{\+:x\}"{}, a); // prints "{}1"{}. fmt\+::print("{}\{\+:b\}"{}, b); // prints "{}01000"{}. fmt\+::print("{}\{\+:x\}"{}, b); // prints "{}8"{}.

However, it does not affect the information bit position string representation, e.\+g. fmt\+::print("{}\{\+:n\}"{}, a); // prints "{}0"{}. fmt\+::print("{}\{\+:n\}"{}, b); // prints "{}1"{}.
