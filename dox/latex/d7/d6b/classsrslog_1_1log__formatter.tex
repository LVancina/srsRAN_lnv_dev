\doxysubsection{Detailed Description}
This is the base class that provides a common framework to format log entries to different kinds of formats. User should implement two different kinds of formats\+: a) Basic log entry formatting. b) Generic context formatting.

For context formatting, callbacks are provided so that derived classes handle specific formatting rules.
