Contents of HARQ, excluding any CRC. 

~\newline
 Example\+: If the number of HARQ bits is 20, then it is represented as\+: \mbox{[} HARQ\+\_\+bit\+\_\+19 ... HARQ\+\_\+bit\+\_\+0 \mbox{]} =\texorpdfstring{$>$}{>} \mbox{[} MSB ... LSB \mbox{]}. NOTE\+: If {\ttfamily is\+\_\+valid} == true, then the HARQs bits set in {\ttfamily payload} should be interpreted as an ACK (if the bit is 1) or NACK (if the bit is 0). If {\ttfamily is\+\_\+valid} == false, then all HARQs bits set in {\ttfamily payload} should be interpreted as not-\/detected; note that these bits cannot be ignored when {\ttfamily is\+\_\+valid} == false, as the "{}not-\/detected"{} outcome will be used to acknowledge (negatively) the HARQ processes corresponding to each single bit.
