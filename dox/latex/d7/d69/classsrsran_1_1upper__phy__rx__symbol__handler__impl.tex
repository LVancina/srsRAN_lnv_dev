\doxysubsection{Detailed Description}
Implementation of the upper PHY handler of receive symbols events. 

In this implementation, handling the newly received symbol event takes the following steps.
\begin{DoxyEnumerate}
\item Get an uplink processor from the pool.
\item Call the corresponding method from the uplink processor based on the type of the received symbol (PRACH, SRS, PUSCH). 
\end{DoxyEnumerate}
