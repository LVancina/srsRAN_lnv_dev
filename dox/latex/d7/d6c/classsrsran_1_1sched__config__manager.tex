\doxysubsection{Detailed Description}
Class that handles the creation/reconfiguration/deletion of cell and UE configurations in the scheduler.

In particular, this class is responsible for\+:
\begin{DoxyItemize}
\item notifying back to the MAC when the configurations are complete,
\item adding/removing UE entries in the scheduler reported metrics,
\item validating the configuration requests provided by the MAC. 
\end{DoxyItemize}
