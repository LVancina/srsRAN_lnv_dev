{\footnotesize\ttfamily unsigned srsran\+::re\+\_\+measurement\+\_\+dimensions\+::nof\+\_\+slices = 0}



Number of slices. 

A slice consists of a set of contiguous REs that spans {\ttfamily nof\+\_\+subc} subcarriers and {\ttfamily nof\+\_\+symbols} OFDM symbols. Typically, for a PHY channel, it represents all the REs corresponding to a single Rx antenna port (before channel equalization) or to a single Tx layer (after channel equalization).
