\doxysubsection{Detailed Description}
This class implements a scheduler that pre-\/reserve the PUCCH resources\textquotesingle{} REs on the scheduler grid, to prevent the PUSCH allocator from using them. The pre-\/reserved REs are hereby referred as PUCCH guardbands.
