class srsran\+::resource\+\_\+grid\+\_\+request\+\_\+pool$<$ Grid\+Type $>$}
Thread-\/safe resource grid request pool. 

Contains a pool of requests of slot points and resource grids indexed by slots. A request consist of a slot point a resource grid. The pool access is thread-\/safe and it is done by exchange requests.
