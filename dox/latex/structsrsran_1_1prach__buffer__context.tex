\doxysubsection{Detailed Description}
Collects PRACH time and frequency mapping parameters. 

It contains the necessary parameters for the lower PHY to capture and demodulate the PRACH sequences. Also, it provides the upper PHY with the necessary information to perform the sequence detection.
