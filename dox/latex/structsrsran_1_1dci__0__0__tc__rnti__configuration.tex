Parameter $N_{\textup{UL\_hop}}$, as per TS38.\+212 Section 7.\+3.\+1.\+1.\+1. 

$N_{\textup{UL\_hop}}$ is the number of bits used in the DCI payload to pack the frequency hopping offset, represented by \doxylink{structsrsran_1_1dci__0__0__tc__rnti__configuration_a2442ce5fbec404ab12cb0d1b74ee1696}{hopping\+\_\+offset}.

\begin{DoxyItemize}
\item 1, if $N_{RB}^{UL,BWP}$ is smaller than 50.
\item 2 otherwise.
