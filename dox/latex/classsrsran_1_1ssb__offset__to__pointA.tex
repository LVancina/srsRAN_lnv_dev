\doxysubsection{Detailed Description}
Data type used to represent the frequency offset between Point A and the lowest subcarrier of the lowest common resource block that overlaps with the SS/\+PBCH block. 

The quantity expressed in terms of this data type corresponds to {\itshape offset\+To\+PointA} in TS38.\+211 Section 4.\+4.\+4.\+2 and it is used in TS38.\+211 Section 7.\+4.\+3.\+1 to compute $N_{CRB}^{SSB}$. It is measured as a number of resource blocks, assuming\+:
\begin{DoxyItemize}
\item 15k\+Hz SCS for FR1, and
\item 60k\+Hz SCS for FR2.
\end{DoxyItemize}

The range of values is \{0, ..., 2199\} --- see TS38.\+331 Section 6.\+3.\+2, Information Element {\itshape Frequency\+Info\+DL-\/\+SIB}.

The reader is referred to \href{https://www.sharetechnote.com/html/5G/5G_ResourceBlockIndexing.html}{\texttt{ this page}} and \href{http://nrexplained.com/rbs}{\texttt{ this page}} for more details about resource block indexing.
