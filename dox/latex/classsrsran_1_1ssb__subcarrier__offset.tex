\doxysubsection{Detailed Description}
Data type used to represent the offset from subcarrier zero in common resource block $N_{CRB}^{SSB}$ to subcarrier zero of the SS/\+PBCH block. 

The quantity expressed in terms of this data type corresponds to $ k_{SSB} $ in TS38.\+211 Section 7.\+4.\+3.\+1. It is measured as a number of subcarriers of the same spacing as the SS/\+PBCH block.

For numerology $\mu\in\{0,1\}$\+:
\begin{DoxyItemize}
\item the range is \{0, ..., 23\}, and
\item the reference SS/\+PBCH subcarrier spacing is 15 k\+Hz.
\end{DoxyItemize}

For numerology $\mu\in\{3,4\}$\+:
\begin{DoxyItemize}
\item the range is \{0, ..., 11\}, and
\item the reference SS/\+PBCH subcarrier spacing is provided by the higher layers parameter {\itshape sub\+Carrier\+Spacing\+Common} (see TS38.\+331 Section 6.\+2.\+2, Information Element {\itshape MIB}).
\end{DoxyItemize}
