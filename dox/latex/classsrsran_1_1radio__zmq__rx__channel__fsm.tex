Checks if the waiting time for request or data has expired. 

The request and data wait expires only if\+:
\begin{DoxyItemize}
\item the current state is waiting for a request; or
\item the current state is waiting for data.
\end{DoxyItemize}

The expiration occurs when a certain time had past since\+:
\begin{DoxyItemize}
\item the last state transition from waiting data to waiting for request; or
\item last time an expiration was detected. 
\end{DoxyItemize}
