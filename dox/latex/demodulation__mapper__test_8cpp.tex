\doxysubsection{Detailed Description}
Demodulation mapper unit test. 

The test takes as input vectors containing noisy modulated symbols and the corresponding noise variances. The symbols are demodulated and the resulting bits (both soft and hard versions) are compared with the expected values, also provided by test vectors.
