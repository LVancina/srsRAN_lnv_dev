\doxysubsection{Detailed Description}
Uplink processor main interface. 

Provides a single access point to all uplink-\/related interfaces, that is \doxylink{classsrsran_1_1uplink__processor__notifier}{uplink\+\_\+processor\+\_\+notifier}, uplink\+\_\+processor\+\_\+request\+\_\+handler and \doxylink{classsrsran_1_1uplink__processor__baseband}{uplink\+\_\+processor\+\_\+baseband}.

\begin{DoxyNote}{Note}
All implementations of this interface must trigger an assertion error if the uplink processor is not connected to a notifier before calling \doxylink{classsrsran_1_1lower__phy__uplink__processor_a443ef8da1db54da5df16b36f9d04a3a9}{get\+\_\+prach\+\_\+request\+\_\+handler()}, \doxylink{classsrsran_1_1lower__phy__uplink__processor_a565ed526753d8038e1384b422948b076}{get\+\_\+puxch\+\_\+request\+\_\+handler()} or \doxylink{classsrsran_1_1lower__phy__uplink__processor_a467145dc007edbc91a2a8df090f7a699}{get\+\_\+baseband()}. 
\end{DoxyNote}
