\doxysubsection{Detailed Description}
PDx\+CH processor main interface. 

Provides a single access point to all downlink channel processor interfaces, that is \doxylink{classsrsran_1_1pdxch__processor__notifier}{pdxch\+\_\+processor\+\_\+notifier}, \doxylink{classsrsran_1_1pdxch__processor__request__handler}{pdxch\+\_\+processor\+\_\+request\+\_\+handler} and \doxylink{classsrsran_1_1pdxch__processor__baseband}{pdxch\+\_\+processor\+\_\+baseband}.

\begin{DoxyNote}{Note}
All implementations of this interface must trigger an assertion error if the PDx\+CH processor is not connected to a notifier before calling \doxylink{classsrsran_1_1pdxch__processor_ae5f731dece159954e55a75196df2862b}{get\+\_\+request\+\_\+handler()} or \doxylink{classsrsran_1_1pdxch__processor_a65d0327b5c0fc4b6eea2cb816ba19fc1}{get\+\_\+baseband()}. 
\end{DoxyNote}
