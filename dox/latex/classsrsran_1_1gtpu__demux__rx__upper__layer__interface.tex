\doxysubsection{Detailed Description}
This interface is used to push incoming GTP-\/U messages to the demuxer. 

The GTP-\/U demux component will only be relevant for the reception and de-\/multiplexing of GTP-\/U packets. It does not require Tx capabilities as the corresponding GTP-\/U entities will already be calling the corresponding UDP network gateway.
