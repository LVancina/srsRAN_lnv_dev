\doxysubsection{Detailed Description}
\doxylink{classBase}{Base} class used for transmitting RLC bearers. It provides interfaces for the RLC bearers, for both higher layers and lower layers It also stores interfaces for the higher layers, both for the user-\/plane and the control plane.
