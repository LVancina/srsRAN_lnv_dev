{\footnotesize\ttfamily \mbox{\hyperlink{classunsigned}{unsigned}} srsran\+::dci\+\_\+0\+\_\+0\+\_\+c\+\_\+rnti\+\_\+configuration\+::hopping\+\_\+offset}



Frequency hopping offset, selected from the RRC parameter {\itshape frequency\+Hopping\+Offset\+Lists}. 

{\itshape frequency\+Hopping\+Offset\+Lists}, given by TS38.\+331 Section 6.\+3.\+2, {\itshape PUSCH-\/\+Config} Information Element, provides 2 or 4 selectable frequency offsets, as per TS38.\+214 Section 6.\+3. The {\ttfamily hopping\+\_\+offset} field acts as an index for {\itshape frequency\+Hopping\+Offset\+Lists}, determining which of the possible offsets is used.

If frequency hopping is enabled via the \doxylink{structsrsran_1_1dci__0__0__c__rnti__configuration_ac8c4d11eaaf31f89cebe743ace00d5c7}{frequency\+\_\+hopping\+\_\+flag}, it must be set to one of the following values\+:
\begin{DoxyItemize}
\item (0, 1), if {\itshape frequency\+Hopping\+Offset\+Lists} has 2 possible offsets.
\item (0, 1, 2, 3), if {\itshape frequency\+Hopping\+Offset\+Lists} has 4 possible offsets. 
\end{DoxyItemize}\Hypertarget{structsrsran_1_1dci__0__0__c__rnti__configuration_a88778348f56fa502ef4219abf5283e7d}\label{structsrsran_1_1dci__0__0__c__rnti__configuration_a88778348f56fa502ef4219abf5283e7d}

{\footnotesize\ttfamily \mbox{\hyperlink{classunsigned}{unsigned}} srsran\+::dci\+\_\+0\+\_\+0\+\_\+c\+\_\+rnti\+\_\+configuration\+::\+N\+\_\+ul\+\_\+hop}



Parameter $N_{\textup{UL\_hop}}$, as per TS38.\+212 Section 7.\+3.\+1.\+1.\+1. 

$N_{\textup{UL\_hop}}$ is the number of bits used in the DCI payload to pack the frequency hopping offset, represented by \doxylink{structsrsran_1_1dci__0__0__c__rnti__configuration_ac7b07939c24f137abcd9755ed43e6893}{hopping\+\_\+offset}.

If frequency hopping is enabled via the \doxylink{structsrsran_1_1dci__0__0__c__rnti__configuration_ac8c4d11eaaf31f89cebe743ace00d5c7}{frequency\+\_\+hopping\+\_\+flag}, it must be set to\+:
\begin{DoxyItemize}
\item 1, if the RRC parameter {\itshape frequency\+Hopping\+Offset\+Lists} has 2 possible offsets.
\item 2, if the RRC parameter {\itshape frequency\+Hopping\+Offset\+Lists} has 4 possible offsets.
