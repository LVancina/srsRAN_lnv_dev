\doxysubsection{Detailed Description}
Lower physical layer PDx\+CH processor -\/ Baseband interface. 

Processes baseband samples with OFDM symbol granularity. The OFDM symbol size is inferred from the slot numerology.

Processes a baseband OFDM symbol. 

Modulates the OFDM symbol indicated by {\ttfamily context} and stores the baseband samples in {\ttfamily samples}. If the resource grid is not available at the time of calling this method, the symbol processing is skipped, and the sample buffer is not modified.
