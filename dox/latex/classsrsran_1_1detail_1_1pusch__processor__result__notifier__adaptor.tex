\doxysubsection{Detailed Description}
Adapts the PUSCH processor result notifier to the upper PHY receive results notifier. 

It collects the Channel State Information (CSI), control and data decoding information from a PUSCH processor object and notifies them through the upper PHY notification interface.

\begin{DoxyRemark}{Remarks}
The order of the calls matters. Method on\+\_\+csi() must be called before on\+\_\+uci() or on\+\_\+sch(). Otherwise, an assertion is triggered. 
\end{DoxyRemark}
