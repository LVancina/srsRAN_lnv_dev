\subsubsection*{template$<$typename T$>$\newline
class srsran\+::optional$<$ T $>$}
Optional objects. 

An instance of this class may either contain a value of type {\ttfamily T} (and {\ttfamily \doxylink{classsrsran_1_1optional_a2f160539a76c3f4dd0b50b339b81202e}{has\+\_\+value()} == true}) or be empty (and {\ttfamily \doxylink{classsrsran_1_1optional_a2f160539a76c3f4dd0b50b339b81202e}{has\+\_\+value()} == false}).

The class handles the safe construction, destruction, move and copy of the contained object.
