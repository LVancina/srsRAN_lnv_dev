std::array<uint8_t, NOF_BEAMS> srsran::ssb_configuration::beam_ids

The n-th element of the array indicates what Beam ID to use for the n-th SSB occasion in ssb_bitmap. Only relevant if the n-th bit of ssb_bitmap is set.

◆ offset_to_point_A

ssb_offset_to_pointA srsran::ssb_configuration::offset_to_point_A

Represents the offset to Point A in PRBs as per TS38.331 Section 6.3.2 IE offsetToPointA. Possible values: {0, ..., 2199}.

◆ pss_to_sss_epre

ssb_pss_to_sss_epre srsran::ssb_configuration::pss_to_sss_epre

PSS EPRE to SSS EPRE for SSB. TS 38.213, Section 4.1 gives an explanation of this measure, but doesn't provide a name for this parameter. Nor is there a name in the TS 38.331.

◆ ssb_bitmap

uint64_t srsran::ssb_configuration::ssb_bitmap

Each bit in this bitmap represents if a beam is active or not as per TS38.331 Section 6.3.2 IE ssb-PositionsInBurst.

◆ ssb_block_power

int srsran::ssb_configuration::ssb_block_power

Average EPRE of the resources elements that carry secondary synchronization signals in dBm, as per ServingCellConfigCommonSIB, TS 38.331. Possible values: {-60, ..., 50}
