Detailed Description

Data type used to represent the frequency offset between Point A and the lowest subcarrier of the lowest common resource block that overlaps with the SS/PBCH block.

The quantity expressed in terms of this data type corresponds to offsetToPointA in TS38.211 Section 4.4.4.2 and it is used in TS38.211 Section 7.4.3.1 to compute . It is measured as a number of resource blocks, assuming:

  • 15kHz SCS for FR1, and
  • 60kHz SCS for FR2.

The range of values is {0, ..., 2199} — see TS38.331 Section 6.3.2, Information Element FrequencyInfoDL-SIB.

The reader is referred to this page and this page for more details about resource block indexing.
