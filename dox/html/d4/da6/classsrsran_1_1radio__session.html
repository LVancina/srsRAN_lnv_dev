◆ start()

virtual void srsran::radio_session::start ( baseband_gateway_timestamp init_time)
pure virtual

Starts the radio operation.

It requests all radio streams to start simultaneously.

Parameters
[in]init_timeInitial time in which streaming shall start.
Remarks
Radio streams run until halted by a call of the stop() method.
Calling start() more than once results in undefined behavior, even after calling stop().

Implemented in srsran::radio_session_uhd_impl, and srsran::radio_session_zmq_impl.

◆ stop()

virtual void srsran::radio_session::stop ( )
pure virtual

Stops the radio session operation.

Any call to radio operation after calling stop() will return instantly without interacting with the physical radio.

Remarks
Calling stop() without having called start() results in undefined behavior.
Calling start() after stop() results in undefined behavior.

Implemented in srsran::radio_session_uhd_impl, and srsran::radio_session_zmq_impl.
