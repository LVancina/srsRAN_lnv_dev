Detailed Description

Data type used to represent the offset from subcarrier zero in common resource block to subcarrier zero of the SS/PBCH block.

The quantity expressed in terms of this data type corresponds to in TS38.211 Section 7.4.3.1. It is measured as a number of subcarriers of the same spacing as the SS/PBCH block.

For numerology :

  • the range is {0, ..., 23}, and
  • the reference SS/PBCH subcarrier spacing is 15 kHz.

For numerology :

  • the range is {0, ..., 11}, and
  • the reference SS/PBCH subcarrier spacing is provided by the higher layers parameter subCarrierSpacingCommon (see TS38.331 Section 6.2.2, Information Element MIB).

The reader is referred to this page and this page for a more details about resource block indexing.
