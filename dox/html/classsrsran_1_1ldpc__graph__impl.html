uint16_t srsran::ldpc_graph_impl::get_lifted_node ( unsigned m,
unsigned n ) const
inline

Returns the compact representation of the lifted node at position .

If the base graph has an edge between check node m and variable node n, the returned value is the shift applied to the lifted node. Conversely, if there is no edge, then the returned value is srsran::ldpc::NO_EDGE.
