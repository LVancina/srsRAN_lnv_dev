bool srsran::ptrs_uplink_config::transform_precoder_enabled::is_t_density_trans_precoding_d2 {false}

Time density (OFDM symbol level) of PT-RS for DFT-s-OFDM. Mapping to NR RRC: If true, the field is present in NR RRC indicating value d2 to UE. If false, the field is absent in NR RRC and the UE applies value d1.

◆ sampl_density

static_vector<uint16_t, max_nof_ptrs_dft_ofdm_sample_density> srsran::ptrs_uplink_config::transform_precoder_enabled::sampl_density

Sample density of PT-RS for DFT-s-OFDM, pre-DFT, indicating a set of thresholds T={NRBn, n=0,1,2,3,4}, that indicates dependency between presence of PT-RS and scheduled BW and the values of X and K the UE should use depending on the scheduled BW. See TS 38.214, clause 6.1, table 6.2.3.2-1. Value {1..276}.
