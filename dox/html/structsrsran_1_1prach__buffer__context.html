Detailed Description

Collects PRACH time and frequency mapping parameters.

It contains the necessary parameters for the lower PHY to capture and demodulate the PRACH sequences. Also, it provides the upper PHY with the necessary information to perform the sequence detection.

unsigned srsran::prach_buffer_context::nof_fd_occasions

Number of PRACH frequency-domain occasions for each of the time-domain occasion.

Corresponds to the higher layer parameter msg1-FDM (TS38.331 Section 6.3.2, Information Element RACH-ConfigGeneric). Possible values are 1, 2, 4 and 8.

◆ nof_td_occasions

unsigned srsran::prach_buffer_context::nof_td_occasions

Number of PRACH time-domain occasions within the slot.

Corresponds to parameter as per TS38.211 Section 5.3.2. It is selected from TS38.211 Tables 6.3.3.2-2, 6.3.3.2-3 and 6.3.3.2-4.

Possible values are from one to seven. Set to one for long preambles.
