Detailed Description

UE Transmit Power Control (TPC) command configuration for SRS.

The identifiers are inspired by TS38.331 Section as 6.3.2 (see field SRS-TPC-CommandConfig).

Indicate whether or not an SRS Request will be included i.e. 0 means SRS request bits in the field is 0 bits and 1 means SRS request bits in the field is 2 bits. Values {0..1}.

◆ starting_bit_of_format_2_3

optional<uint8_t> srsran::srs_tpc_command_config::starting_bit_of_format_2_3

The starting bit position of a block within the group DCI with SRS request fields (optional) and TPC commands. The value 1 of the field corresponds to the first/left most bit of format2-3. See TS 38.212, clause 7.3.1 and TS 38.213, clause 11.3. Values {1..31}.
