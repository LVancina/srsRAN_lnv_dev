Detailed Description

PDxCH processor main interface.

Provides a single access point to all downlink channel processor interfaces, that is pdxch_processor_notifier, pdxch_processor_request_handler and pdxch_processor_baseband.

Note
All implementations of this interface must trigger an assertion error if the PDxCH processor is not connected to a notifier before calling get_request_handler() or get_baseband().
