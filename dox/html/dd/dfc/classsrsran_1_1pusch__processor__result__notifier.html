Detailed Description

PUSCH processor result notifier.

The PUSCH processor uses this interface for notifying results.

virtual void srsran::pusch_processor_result_notifier::on_uci ( const pusch_processor_result_control & uci)
pure virtual

Notifies the detection of Uplink Control Information (UCI) on PUSCH.

The UCI result is always notified after on_csi(), if the pusch_processor::uci_description contains any payload bits. Otherwise, it is not notified.
