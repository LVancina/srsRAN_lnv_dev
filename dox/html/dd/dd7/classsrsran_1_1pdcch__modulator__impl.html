Modulates a PDCCH codeword according to TS38.211 Section 7.3.2.

Parameters
[out]mapperResource grid mapper interface.
[in]dataPDCCH codeword to modulate.
[in]configNecessary parameters for modulating PDCCH.
Remarks
The codeword length shall be consistent with the resource mapping.
An assertion is triggered if the number of precoding layers is not one.
