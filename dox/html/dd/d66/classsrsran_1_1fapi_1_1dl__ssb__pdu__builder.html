Use this function when the PHY generates the timing PBCH information.
These parameters are specified in SCF-222 v4.0 section 3.4.2.4, in table MAC generated MIB PDU.
This function assumes that given bch_payload value is codified as: 0,0,0,0,0,0,0,0,a0,a1,a2,...,a21,a22,a23, with the most significant bit being the leftmost (in this case a0 in position 24 of the uint32_t).
