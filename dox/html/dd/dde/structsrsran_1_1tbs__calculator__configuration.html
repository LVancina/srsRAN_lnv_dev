Detailed Description

Collection of parameters required to calculate the TBS for a shared channel transmission.

The configuration applies to the determination of the transport block size (TBS) for both PDSCH, as described in TS38.214 Section 5.1.3.2, and PUSCH, as described in TS38.214 Section 6.1.4.2.

◆ mcs_descr

sch_mcs_description srsran::tbs_calculator_configuration::mcs_descr

Modulation and coding scheme for the SCH transmission.

The modulation scheme yields the number of bits per RE, parameter for both PDSCH and PUSCH. Possible values are:

  • QPSK,
  • QAM-16,
  • QAM-64, and
  • 256-QAM.

◆ nof_layers

unsigned srsran::tbs_calculator_configuration::nof_layers

Number of layers.

Parameter . For PDSCH, the possible values are:

  • {1, ..., 4} for one codeword, and
  • {5, ..., 8} for two codewords.

For PUSCH, which only admits single-codeword transmissions, the only possible values are {1, ..., 4}.

◆ nof_oh_prb

unsigned srsran::tbs_calculator_configuration::nof_oh_prb

Number of REs configured as overhead.

Parameter . Possible values are {0, 6, 12, 18}.

Specifically, for PDSCH, the value is zero if:

  • the PDSCH is scheduled by PDCCH with a CRC scrambled by SI-RNTI, RA-RNTI or P-RNTI, or
  • the higher layer parameter xOverhead in PDSCH-ServingCellconfig is not configured.

Otherwise, the value is provided by the parameter xOverhead in PDSCH-ServingCellconfig.

Similarly, for PUSCH, the value is zero if:

  • the transport block corresponds to a Msg3 transmission, or
  • the higher layer parameter xOverhead in PUSCH-ServingCellConfig is not configured.

Otherwise, the value is provided by the parameter xOverhead in PUSCH-ServingCellConfig.

◆ tb_scaling_field

unsigned srsran::tbs_calculator_configuration::tb_scaling_field

TB scaling field.

Provides the scaling factor according to TS38.214 Table 5.1.3.2-2. Possible values are {0, 1, 2}.

For the PDSCH assigned by a PDCCH with DCI format 1_0 with CRC scrambled by P-RNTI or RA-RNTI, the value is provided by the TB scaling field of the DCI. For other PDSCHs, it must be set to zero.

For PUSCH, the parameter is meaningless and must be set to zero.
