Detailed Description

Resource grid writer interface.

Contains the necessary functions to write resource elements in a resource grid.

Remarks
All the methods contained in this interface must not result in writing outside of the resource grid region.

Puts a number of resource elements for the given port and symbol l, starting at k_init and at a distance of stride.

Parameters
[in]portPort index.
[in]lSymbol index.
[in]k_initInitial subcarrier index.
[in]strideDistance between adjacent symbols. A stride of 1 means that the allocated REs are contiguous.
[in]symbolsSymbols to be written into the resource grid.
Note
The RE positions given k_init, the number of elements in symbols and the stride shall be within the resource grid number of subcarriers.
