Detailed Description

User interface of the Uplink Shared Channel (UL-SCH) demultiplexer.

Note that, for calculating the CSI Part 2 report size, it is necessary to decode the CSI Part 1. Consequently, ulsch_demultiplex::set_csi_part2 must be called with the size information obtained from CSI Part 1 before the CSI Part 2 report can be demultiplexed.

All fields can be demultiplexed simultaneously if CSI Part 2 is not present (see ulsch_demultiplex::demultiplex).
