static void set_seed (uint32_t seed)
 Set random seed for the test pseudo-random generator. If this function is never called, the pseudo-random generator will be initialized with a random seed using std::random_device. If it is called once, the seed will be the one passed as an argument. If it is called more than once, the test will abort. Thus, there should not be race conditions in seed initialization.
