Detailed Description

template<typename measure_type, unsigned MAX_NOF_SUBC, unsigned MAX_NOF_SYMBOLS, unsigned MAX_NOF_SLICES>
class srsran::static_re_measurement< measure_type, MAX_NOF_SUBC, MAX_NOF_SYMBOLS, MAX_NOF_SLICES >

Static container for RE measurements.

The amount of memory reserved is fixed and set at compilation time.

Template Parameters
measure_typeThe type of the represented RE measurement (e.g., float or cf_t).
MAX_NOF_SUBCMaximum number of subcarriers.
MAX_NOF_SYMBOLSMaximum number of symbols.
MAX_NOF_SLICESMaximum number of slices.
