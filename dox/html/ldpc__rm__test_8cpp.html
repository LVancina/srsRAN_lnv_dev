Detailed Description

LDPC rate matching and recovering unit test.

Test vectors are provided that contain full-length codeblocks and the corresponding rate-matched ones, together with all rate matcher configuration parameters. The test compares the examples with internally generated versions of the codeblocks (for both rate-matching and rate-dematching). The test vectors cover a sufficiently wide range of configuration parameters.
