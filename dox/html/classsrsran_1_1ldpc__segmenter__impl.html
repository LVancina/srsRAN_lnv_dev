Detailed Description

Generic implementation of LDPC segmentation.

Implements both ldpc_segmenter_tx and ldpc_segmenter_rx. For this reason, the constructor has been hidden behind the static wrapper methods create_ldpc_segmenter_impl_tx() and create_ldpc_segmenter_impl_rx().

Splits a codeword into codeblocks and computes the metadata of all codeblocks.

The LLRs corresponding to the received codeword are split into codeblocks to feed the decoder. Each codeblock is described by all the metadata needed for rate-dematching and decoding, as per TS38.212 Section 5.4.2.1.

Parameters
[out]described_codeblocksCodeblock LLRs and corresponding metadata.
[in]codeword_llrsLog-likelihood ratios of the received codeword.
[in]tbsTransport block size (not including CRC), as a number of bits.
[in]cfgParameters affecting splitting and codeblock metadata.
Remarks
The output codeblocks are just views (not copies) of the proper block of codeword log-likelihood ratios in codeword_llrs.

overridevirtual

Adds the CRC to the a transport block, carries out segmentation and computes all codeblock metadata for later use (encoder and rate matching).

First, the transport block CRC is attached, as per TS38.212 Section 7.2.1. Then, the transport block is split into a number of segments and, if needed, a CRC is attached to each segment_tx. This is done according to TS38.212 Section 5.2.2. The function also computes other segment_tx metadata (e.g., coded and rate-matched length) according to TS38.212 Section 5.4.2.1.

Parameters
[out]described_segmentsSegments (unpacked, one bit per entry) and corresponding metadata.
[in]transport_blockThe transport block to segment_tx (packed, one byte per entry).
[in]cfgParameters affecting splitting and codeblock metadata.
