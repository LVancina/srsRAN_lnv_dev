Detailed Description

PUCCH processor interface for all formats.

Remarks
The total payload size for Formats 2, 3 and 4 shall not exceed 1706. Specifically, one should have nof_sr + nof_harq_ack + nof_csi_part1 + nof_csi_part2 <= 1706.
