Detection, decoding and rate-dematching of short blocks.

Checks whether the provided signal contains a codeblock and, if so, decodes its content assuming the encoding followed TS38.212 Section 5.3.3 and the rate-matching scheme of TS38.212 Section 5.4.3.

Parameters
[out]outputDetected message (one bit per entry).
[in]inputReceived codeblock (log-likelihood ratios).
[in]modModulation scheme.
Returns
True if input passes a detection test.
Remarks
When the detection test fails, the function returns false and the content of output is unreliable.
