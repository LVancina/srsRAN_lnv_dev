Convolution between two sequences.

Computes the convolution between two sequences and , namely and returns its central part, of the same size as the first input sequence.

Template Parameters
VOutput type (must be compatible with a span of arithmetic/complex values).
TType of the first input sequence (must be compatible with a span of arithmetic/complex values).
UType of the second input sequence (must be compatible with a span of arithmetic/complex values).
Parameters
[out]outOutput sequence - convolution between the input sequences.
[in]x_vFirst input sequence.
[in]y_vSecond input sequence.
Remarks
An assertion is raised if the length of the second sequence is larger than the length of the first one.
This implementation of the convolution is not based on FFT and should only be used when the second input sequence is "very" short.
