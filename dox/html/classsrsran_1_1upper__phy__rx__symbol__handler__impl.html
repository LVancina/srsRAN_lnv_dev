Detailed Description

Implementation of the upper PHY handler of receive symbols events.

In this implementation, handling the newly received symbol event takes the following steps.

  1. Get an uplink processor from the pool.
  2. Call the corresponding method from the uplink processor based on the type of the received symbol (PRACH, SRS, PUSCH).
