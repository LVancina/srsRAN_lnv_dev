Detailed Description

Describes a PUSCH rate matcher buffer.

The aim is to provide the physical layer shared channel FEC with a persistent storage of CRC and soft bits buffer among transmissions for a given user and HARQ process.

The CRCs are set to true if the codeblocks have been successfully decoded. By looking at these flags, codeblocks decoded on a prior transmission do not require decoding.

It is the responsibility of the shared channel receiver to reset CRCs and set to zero the soft bits when new data is indicated.
