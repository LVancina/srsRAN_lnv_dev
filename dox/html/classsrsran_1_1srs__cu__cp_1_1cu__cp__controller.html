Detailed Description

Entity responsible for managing the CU-CP connections to remote nodes and determining whether the CU-CP is in a state to accept new connections.

In particular, this class is responsible for:

  • triggering an AMF connection and tracking the status of the connection;
  • determining whether a new DU setup request should be accepted based on the status of other remote node connections;
  • determining whether new UEs should be accepted depending on the status of the CU-CP remote connections.
