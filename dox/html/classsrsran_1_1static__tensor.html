Detailed Description

template<unsigned NDIMS, typename Type, unsigned MAX_ELEMENTS, typename Index_type = unsigned>
class srsran::static_tensor< NDIMS, Type, MAX_ELEMENTS, Index_type >

Static tensor - the dimensions can be resized without allocating memory up to MAX_ELEMENTS elements.

Template Parameters
NDIMSNumber of dimensions.
TypeType of data to store.
MAX_ELEMENTSmaximum number of elements.
Index_typeData type used for representing dimension indexes. It is used in method get_dimension_size().
