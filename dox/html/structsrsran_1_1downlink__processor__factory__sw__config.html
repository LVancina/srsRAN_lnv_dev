std::string srsran::downlink_processor_factory_sw_config::crc_calculator_type

CRC calculator type.

Use of there options:

  • auto: let the factory select the most efficient given the CPU architecture, or
  • lut: for using a look-up table CRC calculator, or
  • clmul: for using a look-up table CRC calculator (x86_64 CPUs only).

◆ ldpc_encoder_type

std::string srsran::downlink_processor_factory_sw_config::ldpc_encoder_type

LDPC encoder type.

Use of there options:

  • auto: let the factory select the most efficient given the CPU architecture, or
  • generic: for using unoptimized LDPC encoder, or
  • avx2: for using AVX2 optimized LDPC encoder (x86_64 CPUs only), or
  • neon: for using NEON optimized LDPC encoder (ARM CPUs only).

PDSCH processor type.

Use of there options:

  • generic: for using unoptimized PDSCH processing, or
  • concurrent: for using a processor that processes code blocks in parallel, or
  • lite: for using a memory optimized processor.
