Detailed Description

Non-owning view to a byte sequence. The underlying byte sequence is not contiguous in memory. Instead, it is represented as an intrusive linked list of byte buffer segments, where each segment contains a span of bytes.

Returns a non-owning list of segments that compose the byte_buffer. The segments are not const, so that the callee can modify the bytes, but not layout of the buffer.
