Detailed Description

This class implements a scheduler that pre-reserve the PUCCH resources' REs on the scheduler grid, to prevent the PUSCH allocator from using them. The pre-reserved REs are hereby referred as PUCCH guardbands.

void pucch_guardbands_scheduler::run_slot ( cell_resource_allocator & res_alloc)

Runs the PUCCH guardbands scheduler.

When called for the first time, this function pre-reserves the PUCCH's REs over the entire grid, until the last (farthest in the future) usable slot. From the second time onwards, the scheduler pre-reserves the PUCCH for the last usable slot of the grid.
