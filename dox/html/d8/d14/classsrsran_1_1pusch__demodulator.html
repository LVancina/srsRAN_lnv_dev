Detailed Description

PUSCH demodulator interface.

The demodulation of a PUSCH consists of:

  • extracting allocated REs from the resource grid,
  • equalizing of the extracted RE,
  • soft-demodulation of the complex data, and
  • descrambling.

Demodulates a PUSCH transmission.

Computes log-likelihood ratios from channel samples by reversing all the operations described in TS38.211 Section 6.3.1.

Remarks
  • The size of data determines the codeword size in soft bits.
  • The total number of LLR must be consistent with the number of RE allocated to the transmission.
