virtual span< const uint8_t > srsran::ecpri::packet_decoder::decode ( span< const uint8_t > packet,
packet_parameters & params )
pure virtual

Decodes the given packet, filling the eCPRI params and returning a span to the eCPRI payload.

If the decoder detects an unsupported value, it will return an empty span and the params value will be undefined.

Parameters
[in]packeteCPRI packet to decode.
[out]paramseCPRI decoded parameters.
Returns
A span to the eCPRI payload on success, otherwise an empty span in case of error.
