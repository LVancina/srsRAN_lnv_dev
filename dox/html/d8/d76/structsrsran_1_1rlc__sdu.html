Detailed Description

Structure used to represent an RLC SDU. An RLC SDU can optionally be accompanied with the corresponding PDCP sequence number (SN) so that RLC AM can notify the PDCP of ACKs, and PDCP can notify RLC AM/UM to discard PDCP PDUs
