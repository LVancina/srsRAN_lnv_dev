Detailed Description

Describes CORESET parameters.

Provides the Control Resource Set (CORESET, TS38.331 ControlResourceSet for more information) related parameters common for up to MAX_NOF_DCI Downlink Control Information (DCI) transmissions.

For controlResourceSetZero (CORESET 0, TS38.213 Section 13 for more information) there are some exceptions:

  • cce_to_reg_mapping_type shall be set to CORESET0.
  • bwp_start_rb and bwp_size_rb indicate the CORESET start point and contiguous resource blocks.
  • shift_index shall be equal to the physical cell identifier {0...1007}.
  • cce_to_reg_mapping_type, frequency_resources, reg_bundle_size, interleaver_size are ignored.

Ignored for cce_to_reg_mapping_type set to INTERLEAVED or CORESET0. It must be {2,3,6}.

◆ reg_bundle_size

unsigned srsran::pdcch_processor::coreset_description::reg_bundle_size

The number of REGs in a bundle. Corresponds to parameter in TS38.211 7.3.2.2.

Ignored for cce_to_reg_mapping_type set to INTERLEAVED or CORESET0. It must be:

  • {2,6} for duration of 1 or 2 symbols, and
  • {3,6} for duration of 3 symbols.

◆ shift_index

unsigned srsran::pdcch_processor::coreset_description::shift_index

Shift index in TS38.211 Section 7.3.2.2.

The value is:

  • set to the physical cell identifier for a transmission in CORESET0,
  • ignored for non-interleaved mapping, and
  • set to {0,275} for non-interleaved CCE-to-REG mapping.
