bounded_bitset<uci_constants::MAX_NOF_CSI_PART1_OR_PART2_BITS> srsran::mac_uci_pdu::pusch_type::csi_information::payload

Contents of CSI, excluding any CRC. Example: If the number of CSI bits is 20, then it is represented as: [ CSI_bit_19 ... CSI_bit_0 ] => [ MSB ... LSB ]. NOTE: if is_valid == false, then the CSI payload will be ignored.
