Detailed Description

Uplink processor interface.

The uplink processor is in charge of handling incoming requests to process the physical uplink channels within a certain slot.

Requests are dispatched asynchronously as they get enqueued for execution, and generate an event through the upper_phy_rx_results_notifier interface upon completion.
