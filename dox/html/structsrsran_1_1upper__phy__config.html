◆ crc_calculator_type

std::string srsran::upper_phy_config::crc_calculator_type

CRC calculator type.

Use of there options:

  • auto: let the factory select the most efficient given the CPU architecture, or
  • lut: for using a look-up table CRC calculator, or
  • clmul: for using a look-up table CRC calculator (x86_64 CPUs only).

◆ ldpc_decoder_type

std::string srsran::upper_phy_config::ldpc_decoder_type

LDPC decoder type.

Use of there options:

  • auto: let the factory select the most efficient given the CPU architecture, or
  • generic: for using generic instructions, or
  • avx2: for using AVX2 instructions (x86_64 CPUs only), or
  • avx512: for using AVX512 instructions (x86_64 CPUs only), or
  • neon: for using NEON instructions (ARM CPUs only).
