◆ n_id_pdcch_data

unsigned srsran::dci_context_information::n_id_pdcch_data

Parameter used for PDCCH Data scrambling as per 3GPP TS 38.211 [2], sec 7.3.2.3. Values: (0..65535). For a UE-specific search space it equals the higherlayer parameter PDCCH-DMRS-Scrambling-ID if configured, otherwise it should be set to the phy cell ID.

◆ n_rnti_pdcch_data

unsigned srsran::dci_context_information::n_rnti_pdcch_data

Parameter used for PDCCH data scrambling, as per 3GPP TS 38.211 [2], sec 7.3.2.3. Values: (0..65535). For a UE-specific search space where PDCCH-DMRSScrambling-ID is configured, this param equals the CRNTI. Otherwise, it should be set to 0.
