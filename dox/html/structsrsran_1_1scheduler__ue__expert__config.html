dc_offset_t srsran::scheduler_ue_expert_config::initial_ul_dc_offset {dc_offset_t::center}

Direct Current (DC) offset, in number of subcarriers, used in PUSCH, by default. The gNB may supersede this DC offset value through RRC messaging. See TS38.331 - "txDirectCurrentLocation".

◆ ta_cmd_offset_threshold

int8_t srsran::scheduler_ue_expert_config::ta_cmd_offset_threshold

Timing Advance Command (T_A) offset threshold above which Timing Advance Command is triggered. Possible valid values {0,...,32}. If set to less than zero, issuing of TA Command is disabled.
