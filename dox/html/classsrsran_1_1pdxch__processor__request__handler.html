Detailed Description

Lower physical layer downlink processor - Request handler interface.

This class provides an interface for requesting the modulation of a downlink resource grid.

virtual void srsran::pdxch_processor_request_handler::handle_request ( const resource_grid_reader & grid,
const resource_grid_context & context )
pure virtual

Handles a new downlink resource grid transmission request.

The downlink resource grid transmission and context provided to this method are forwarded to a dedicated block that takes care of the request asynchronously. A notification through the downlink_processor_notifier interface will inform the client if the request has not been processed.
