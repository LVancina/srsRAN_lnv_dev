31/*
32 * This file will hold the interfaces and notifiers for the RLC entity.
33 * They follow the following nomenclature:
34 *
35 * rlc_{tx/rx}_{lower/upper}_layer_{[control/data]}_{interface/notifier}
36 *
37 * 1. TX/RX indicates whether the interface is intended for the
38 * TX or RX side of the entity
39 * 2. Lower/Upper indicates whether the interface/notifier interacts
40 * with the upper or lower layers.
41 * 3. Control/Data: indicates whether this interface is necessary for "control"
42 * purposes (e.g., notifying the RRC of max retransmissions, or that we are
43 * near max HFN) or "data" purposes (e.g. handling SDUs).
44 * This distinction is only necessary when interfacing with the upper layers,
45 * and as such, we omit it in the interfaces with the lower layers.
46 * 4. Interface/Notifier: whether this is an interface the RLC entity will
47 * inherit or a notifier that the RLC will keep as a member.
48 *
49 */
