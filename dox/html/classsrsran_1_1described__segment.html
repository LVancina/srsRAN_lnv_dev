Detailed Description

Describes an LDPC segment or codeblock.

The segment description consists of a set of parameters contained in metadata attribute and a data storage for the segment.

If the object is created with the default constructor, the get_data() methods trigger an assertion.

Creates a description for a segment of length cb_size with the given metadata.

This constructor does not initialize the codeblock data. It is up to the class client to fill the codeblock data after its creation.
