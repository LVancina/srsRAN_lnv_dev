byte_buffer_view bit_decoder::unpack_aligned_bytes ( size_t n_bytes)

Create a view across n_bytes bytes of the underlying byte_buffer, starting at the first full byte from the current location. For this purpose, the decoder is first automatically aligned via align_bytes.

Parameters
n_bytesNumber of bytes to read.
Returns
byte_buffer_view across n_bytes bytes of the underlying byte_buffer. An empty view is returned if the number of bytes to be read exceeds the end of the byte_buffer.
