Detailed Description

template<typename simdWrapper, typename storageType>
class srsran::detail::simd_span< simdWrapper, storageType >

Mimics a span of SIMD registers.

Template Parameters
simdWrapperWrapper for an SIMD vector type.
storageTypeUnderlying data type of the simd_span (either int8_t or const int8_t).
