Detailed Description

Describes a PDCCH modulator interface.

Remarks
Defined in TS 38.211 section 7.3.2 Physical downlink control channel (PDCCH).
Note
The configuration assumes the CCE-to-REG and REG-to-PRB mapping is included in rb_mask.
