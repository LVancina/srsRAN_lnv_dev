void ssb_scheduler::run_slot ( cell_resource_allocator & res_allocator,
const slot_point & sl_point )

Schedule grant(s) for SSB.

The functions schedules the SSB according to a given periodicity, depending on the frequency and SSB case for up to N slots when run for first time. In subsequent runs schedules SSB on N + 1 slot.

Parameters
[out]res_allocatorCell resource grid.
[in]sl_pointSlot point carrying information about current slot.
Remarks
This function only works for FR1, or L_max = 4 or 8.
