Detailed Description

Describes a Discrete Fourier Transform (DFT) processor.

Remarks
The DFT processor allocates the input and output buffers for optimization purposes.
The input buffer is available using get_input().
The output buffer is available using run().

Indicates the DFT direction.

Enumerator
DIRECT 

Converts a time domain signal into frequency domain.

Remarks
For the FFTW based libraries, it is equivalent to FFTW_FORWARD sign.
INVERSE 

Converts a frequency domain signal into frequency (inverse DFT).

Remarks
For the FFTW based libraries, it is equivalent to FFTW_BACKWARD sign.
