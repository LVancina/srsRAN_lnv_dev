Detailed Description

This class adapts srsRAN scheduler interface to operate with srsRAN MAC. The configuration completion notification handling (e.g. ue creation complete) is deferred for later processing rather than being processed inline. We defer the processing because we do not want it to take place while the scheduler is still processing the slot_indication, given that latter has higher priority.
