Detailed Description

Used to configure downlink demodulation reference signals for PDSCH.

Remarks
See TS 38.331, DMRS-DownlinkConfig.

◆ additional_positions

dmrs_additional_positions srsran::dmrs_downlink_config::additional_positions {dmrs_additional_positions::pos2}

Position for additional DM-RS in DL, see Tables 7.4.1.1.2-3 and 7.4.1.1.2-4 in TS 38.211. If the field is absent, the UE applies the value pos2.

◆ is_dmrs_type2

bool srsran::dmrs_downlink_config::is_dmrs_type2 {false}

Selection of the DMRS type to be used for DL. Mapping to NR RRC: If true, the field is present in NR RRC indicating value type2 to UE. If false, the field is absent in NR RRC and the UE uses DMRS type 1.

◆ is_max_length_len2

bool srsran::dmrs_downlink_config::is_max_length_len2 {false}

The maximum number of OFDM symbols for DL front loaded DMRS. Mapping to NR RRC: If true, the field is present in NR RRC indicating value len2 to UE. If false, the field is absent in NR RRC and the UE applies value len1. If set to len2, the UE determines the actual number of DM-RS symbols by the associated DCI.

◆ scrambling_id0

optional<uint16_t> srsran::dmrs_downlink_config::scrambling_id0

DL DMRS scrambling initialization (see TS 38.211, clause 7.4.1.1.1). When the field is absent the UE applies the value Physical cell ID (physCellId) configured for this serving cell.
