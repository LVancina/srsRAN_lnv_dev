Builds a VLAN Ethernet frame using the given parameters into buffer.

This function expects that the buffer already contains the payload of the frame. The builder will write the header in the first vlan_ethernet_frame_builder::get_header_size bytes of the buffer.

[in]bufferBuffer where the frame will be built.
