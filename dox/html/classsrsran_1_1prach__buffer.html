Detailed Description

PRACH buffer interface.

Provides access to frequency-domain PRACH sequences.

A buffer for storing long PRACH sequences ( ), for a maximum of 4 OFDM symbols, can be created with the factory function create_prach_buffer_long(). A buffer for storing short PRACH sequences ( ), for a maximum of 12 OFDM symbols, can be created with the factory function create_prach_buffer_short().
