Detailed Description

This interface represents the data exit point of the receiving side of a F1-U bearer of the CU-UP. The F1-U bearer will use this notifier to pass NR-U SDUs (e.g. PDCP PDUs/RLC SDUs) to upper layer (e.g. PDCP).
