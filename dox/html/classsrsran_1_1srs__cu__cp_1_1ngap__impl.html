Initiates the NG Setup procedure.

Parameters
[in]requestThe NGSetupRequest message to transmit.
Returns
Returns a ngap_ng_setup_result struct.
Remarks
The CU transmits the NGSetupRequest as per TS 38.413 section 8.7.1 and awaits the response. If a NGSetupFailure is received the NGAP will handle the failure.
