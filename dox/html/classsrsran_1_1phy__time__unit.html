Detailed Description

Physical layer time unit.

This class abstracts the PHY reference time unit described in TS38.211 Section 4.1.

Determines if the time value is sample accurate given a sampling rate in hertz.

The time value is sample accurate if the time is equivalent to an integer number of samples for the given sampling rate.

Template Parameters

Parameters
[in]sampling_rate_HzSample rate in hertz.
Returns
