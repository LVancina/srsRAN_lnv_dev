Detailed Description

Uplink processor main interface.

Provides a single access point to all uplink-related interfaces, that is uplink_processor_notifier, uplink_processor_request_handler and uplink_processor_baseband.

Note
All implementations of this interface must trigger an assertion error if the uplink processor is not connected to a notifier before calling get_prach_request_handler(), get_puxch_request_handler() or get_baseband().
