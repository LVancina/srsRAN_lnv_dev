Detailed Description

Lower physical layer PUxCH processor - Request handler interface.

Provides an interface for requesting the capture and demodulation of PUxCH occasions.

const resource_grid_context & context )
pure virtual

Handles a new PUxCH request.

The resource grid and context provided to this method are forwarded to a dedicated block that takes care of the request asynchronously. A notification through the puxch_processor_notifier interface will inform the client that the request has been processed and that the PUxCH demodulated sequence is available in grid.
