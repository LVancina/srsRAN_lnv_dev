Detailed Description

Polar code interface.

Defines a polar code based on message length, codeblock length, rate matching and, possibly, internal interleaver. The resulting code is compatible with the specifications in TS38.212 Section 5.3.1.

Provides access to the defining parameters as well as to derived ones.
