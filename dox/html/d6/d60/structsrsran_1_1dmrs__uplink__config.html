Selection of the DMRS type to be used for UL. Mapping to NR RRC: If true, the field is present in NR RRC indicating value type2 to UE. If false, the field is absent in NR RRC and the UE uses DMRS type 1.

◆ is_max_length_len2

bool srsran::dmrs_uplink_config::is_max_length_len2 {false}

The maximum number of OFDM symbols for UL front loaded DMRS. Mapping to NR RRC: If true, the field is present in NR RRC indicating value len2 to UE. If false, the field is absent in NR RRC and the UE applies value len1. If set to len2, the UE determines the actual number of DM-RS symbols by the associated DCI.
