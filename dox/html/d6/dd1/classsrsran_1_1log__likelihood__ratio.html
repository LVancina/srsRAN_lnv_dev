Detailed Description

Represents log-likelihood ratio (LLR) values.

Log-likelihood ratios are represented as int8_t values between -LLR_MAX and LLR_MAX. The values ±LLR_INFTY = ±127 are reserved for denoting fixed bits, that is bits that are surely set to zero (log-likelihood ratio equal to +LLR_INFTY) or one (log-likelihood ratio equal to -LLR_INFTY).
