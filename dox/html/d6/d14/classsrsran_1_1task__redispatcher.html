Detailed Description

template<typename Executor>
class srsran::task_redispatcher< Executor >

Executes a task n times, where "n" increases with the failures to dispatch the task. Dispatches a task to an executor, and if it fails to dispatch (e.g. due to the task queue being full), it reattempts on the next call.
