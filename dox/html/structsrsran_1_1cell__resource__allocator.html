Detailed Description

Circular Ring of cell_slot_resource_grid objects. This class manages the automatic resetting of cell_slot_resource_grid objects, once they become old.

Member Function Documentation

◆ operator[]()

const cell_slot_resource_allocator & srsran::cell_resource_allocator::operator[] ( unsigned slot_delay) const
inline

Access resource allocator for a given {slot, cell}. The accessed slot corresponds to the latest slot indication provided to the cell_resource_grid_manager + slot_delay provided as argument. Given that slot_delay is unsigned, this class can only access the present and future slots.

Parameters
slot_delaydelay in slots added to the last slot indication value.
