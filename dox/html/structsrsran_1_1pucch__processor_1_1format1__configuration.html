unsigned srsran::pucch_processor::format1_configuration::n_id

Parameter in TS38.211 Section 6.3.2.2.1 {0, ..., 1023}.

It must be set to the higher layer parameter hoppingID given by TS38.331 Section 6.3.2, Information Element PUCCH-ConfigCommon, if it is configured. Otherwise, it must be equal to the physical cell identifier .
