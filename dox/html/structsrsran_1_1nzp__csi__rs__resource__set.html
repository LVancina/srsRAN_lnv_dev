Detailed Description

NZP-CSI-RS-ResourceSet is a set of Non-Zero-Power (NZP) CSI-RS resources (their IDs) and set-specific parameters.

Remarks
See TS 38.331, NZP-CSI-RS-ResourceSet.

◆ aperiodic_trigger_offset

optional<unsigned> srsran::nzp_csi_rs_resource_set::aperiodic_trigger_offset

Offset X between the slot containing the DCI that triggers a set of aperiodic NZP CSI-RS resources and the slot in which the CSI-RS resource set is transmitted. Values {0,...,6}.

◆ is_repetition_on

optional<bool> srsran::nzp_csi_rs_resource_set::is_repetition_on

Indicates whether repetition is on/off. If false, UE may not assume that the NZP-CSI-RS resources within the resource set are transmitted with the same downlink spatial domain transmission filter.
