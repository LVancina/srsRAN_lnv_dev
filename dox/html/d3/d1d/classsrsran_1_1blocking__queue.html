Detailed Description

template<typename T, typename PushingCallback = detail::noop_operation, typename PoppingCallback = detail::noop_operation>
class srsran::blocking_queue< T, PushingCallback, PoppingCallback >

Blocking queue with buffer storage represented via a std::vector<T>. Features:

  • Blocking push/pop API via push_blocking(...) and pop_blocking(...) methods
  • Non-blocking push/pop API via try_push(...) and try_pop(...) methods
  • Size can be defined at runtime.
  • thread-safe
    Template Parameters
    Tvalue type stored by buffer
    PushingCallbackfunction void(const T&) called while pushing an element to the queue
    PoppingCallbackfunction void(const T&) called while popping an element from the queue
