Detailed Description

Parameters for PUCCH configuration. Defines the parameters that are used for the PUCCH configuration builder. These parameters are used to define the number of PUCCH resources, as well as the PUCCH format-specific parameters.

◆ nof_cell_harq_pucch_res_sets

unsigned srsran::pucch_builder_params::nof_cell_harq_pucch_res_sets = 1

Number of separate PUCCH resource sets for HARQ-ACK reporting that are available in a cell.

Remarks
UEs will be distributed possibly over different HARQ-ACK PUCCH sets; the more sets, the fewer UEs will have to share the same set, which reduces the chances that UEs won't be allocated PUCCH due to lack of resources. However, the usage of PUCCH-dedicated REs will be proportional to the number of sets.
