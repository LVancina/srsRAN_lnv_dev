virtual bool srsran::resource_grid_reader::is_empty ( ) const
pure virtual

Determines whether the resource grid is empty.

The grid is considered empty if no put method have been called since last zero set.

If zeros are written using any put method, the grid is considered not empty.

Returns
true if the resource grid is empty. Otherwise, false.

Implemented in srsran::resource_grid_reader_empty, srsran::resource_grid_reader_impl, and srsran::ru_dummy_rx_resource_grid.

virtual bool srsran::resource_grid_reader::is_empty ( unsigned port) const
pure virtual

Determines whether a given port is empty.

A port is considered empty if no put method have been called since last zero set.

If zeros are written using any put method, the port is considered not empty.

An invalid port index triggers an assertion.

Parameters
[in]portPort index.
Returns
True if the given port is empty. Otherwise, false.
