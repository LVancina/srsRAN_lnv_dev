◆ last_symbol

bounded_integer<uint8_t, 3, 14> srsran::pdsch_dmrs_symbol_mask_mapping_type_A_single_configuration::last_symbol

Last OFDM symbol scheduled for PDSCH.

It indicates the number of symbols between the first OFDM symbol of the slot and the last OFDM symbol of the scheduled PDSCH resources in the slot for PDSCH in OFDM symbols. It corresponds to parameter as per TS38.211, Section 7.4.1.1.2.

Possible values are {3, ..., 14}.
