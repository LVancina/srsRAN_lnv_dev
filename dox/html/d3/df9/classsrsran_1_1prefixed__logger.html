Detailed Description

template<typename Prefix>
class srsran::prefixed_logger< Prefix >

Class used to store common logging parameters for all types RLC entities. It provides logging helpers, so that the UE index and LCID are always logged.

Parameters
log_namename of the logger we want to use (e.g. RLC, PDCP, etc.)
du_indexUE identifier within the DU
lcidLCID for the bearer
