It reverts the rate matching procedure described in TS38.212 Section 5.4.2.

Recovers a full codeblock from its rate-matched version.

This function also takes care of combining repeated codeblocks for H-ARQ schemes (both chase combining and incremental redundancy).

Log-likelihood ratios corresponding to filler bits are set to INT8_MAX. Log-likelihood ratios that cannot be determined from the input are set to zero.

Parameters
[in,out]outputFull-length codeblock (log-likelihood ratios).
[in]inputRate-matched codeblock (log-likelihood ratios).
[in]cfgConfiguration parameters.
Remarks
The sizes of input and output determine the behavior of the rate recovering algorithm.
