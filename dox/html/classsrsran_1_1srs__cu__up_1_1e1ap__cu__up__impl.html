async_task< cu_up_e1_setup_response > e1ap_cu_up_impl::handle_cu_up_e1_setup_request ( const cu_up_e1_setup_request & request)
overridevirtual

Initiates the E1 Setup procedure as per TS 38.463, Section 8.2.3.

Parameters
[in]requestThe E1SetupRequest message to transmit.
Returns
Returns a cu_up_e1_setup_response struct with the success member set to 'true' in case of a successful outcome, 'false' otherwise.
Remarks
The CU-UP transmits the E1SetupRequest as per TS 38.463 section 8.2.3 and awaits the response. If a E1SetupFailure is received the E1AP will handle the failure.
