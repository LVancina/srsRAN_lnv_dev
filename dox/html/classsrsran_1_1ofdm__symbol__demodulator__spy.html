Demodulates an OFDM signal with symbol granularity.

Parameters
[out]gridProvides the output as frequency-domain signal corresponding to one slot.
[in]inputProvides the time domain modulated signal.
[in]port_indexIndicates the port index to demodulate.
[in]symbol_indexSymbol index within the subframe to demodulate.
Note
The input size must be equal to the the symbol size including the cyclic prefix.
The output size must be consistent with the configured bandwidth.
