Configures the resource grid of the downlink_processor.

Parameters
[in]contextResource grid context that contains the information of the processing slot.
[in]gridResource grid that will contain the data of the processed downlink channels.
Returns
true if the resource grid is successfully configured, false otherwise.
Note
  • Calling this method is mandatory before processing any PDU.
  • The resource grid number of ports and bandwidth must be sufficient to accommodate all the PDUs.

◆ finish_processing_pdus()

void srsran::downlink_processor_spy::finish_processing_pdus ( )
inlineoverridevirtual

Stops accepting PDUs.

When this method is called, the interface will not expect to process more PDUs, so once it finishes to process all the enqueued PDUs, the resource grid will be sent to the lower bound gateway using the context from configure_resource_grid() to provide the processing context of the resource grid in the lower physical layer.
