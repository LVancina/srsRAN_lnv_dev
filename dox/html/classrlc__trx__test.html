Creates a byte_buffer serving as SDU for RLC.

The produced SDU contains an incremental sequence of bytes starting with the value given by first_byte, i.e. if first_byte = 0xfc, the SDU will be 0xfc 0xfe 0xff 0x00 0x01 ...
