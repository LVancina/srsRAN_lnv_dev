Extracts a consecutive count number of bits starting at startpos.

Extracts count bits starting at startpos position and stores them at the count least significant bits.

Template Parameters

Parameters

[in]countNumber of bits to insert.
Returns

Remarks

An assertion is triggered if range of bits exceed the maximum size of the buffer.
