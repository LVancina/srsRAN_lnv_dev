bool srsran::rlc_retx_queue::remove_sn ( uint32_t sn)
inline

Removes all elements with a given SN from queue and removes any invalid elements at the front.

The ReTx for matching SN will be marked as invalid and its content will be subtracted from the state. Subsequently, any invalid elements are removed from the front.

Parameters
snsequence number to be removed from queue.
Returns
true if at least one element was removed, false if no element to remove was found.
