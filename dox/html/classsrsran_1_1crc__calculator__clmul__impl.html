Detailed Description

CRC calculator implementation based on the PCLMULQDQ instruction.

Port of the soft-crc implementation copyrighted by Intel Corporation and licensed under BSD. Source: https://github.com/intel/soft-crc

"Fast CRC Computation for Generic Polynomials Using PCLMULQDQ Instruction" URL: http://download.intel.com/design/intarch/papers/323102.pdf
