unsigned srsran::srsvec::bit_pack ( span< const uint8_t > bits)

Packs the input bits into an integer value.

The first bit in the sequence corresponds to the bit of index bits.size() - 1 of the returned value. The last value in the sequence corresponds to LSB of the returned value.

Parameters
[in]bitsView of unpacked bits.
Returns
An integer containing the packed bits.
Remarks
The number of elements must not exceed 32 bits.

Unpacks an unsigned integer value into bits.

The first bit in the sequence corresponds to the bit of index nof_bits - 1 of value. The LSB of value is mapped to the last element in the sequence.

Parameters
[out]bitsUnpacked bits sequence.
[in]valueProvides the value to unpack.
[in]nof_bitsIndicates the number of bits to unpack.
Returns
A view of the remaining (behind nof_bits) unpacked bits of value.
