◆ nof_slices

unsigned srsran::re_measurement_dimensions::nof_slices = 0

Number of slices.

A slice consists of a set of contiguous REs that spans nof_subc subcarriers and nof_symbols OFDM symbols. Typically, for a PHY channel, it represents all the REs corresponding to a single Rx antenna port (before channel equalization) or to a single Tx layer (after channel equalization).
