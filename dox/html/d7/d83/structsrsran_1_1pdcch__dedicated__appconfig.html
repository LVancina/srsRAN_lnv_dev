std::array<uint8_t, 5> srsran::pdcch_dedicated_appconfig::ss2_n_candidates = {0, 0, 0, 0, 0}

Number of PDCCH candidates per aggregation level for SearchSpace#2. The aggregation level for the array element with index "x" is L=1U << x. The possible values for each element are {0, 1, 2, 3, 4, 5, 6, 8}. NOTE: A value of {0, 0, 0, 0, 0} lets the gNB decide nof. candidates for SearchSpace#2.
