Detailed Description

Lower physical layer PRACH processor - Request handler interface.

This class provides an interface for requesting the capture and demodulation of PRACH occasions.

virtual void srsran::prach_processor_request_handler::handle_request ( prach_buffer & buffer,
const prach_buffer_context & context )
pure virtual

Handles a new PRACH request.

The PRACH buffer and context provided to this method are forwarded to a dedicated block that takes care of the request asynchronously. A notification through the prach_processor_notifier interface will inform the client that the request has been processed and that the PRACH demodulated sequence is available in buffer.
