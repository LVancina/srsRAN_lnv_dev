53/// \brief DCI configuration parameters required to perform the DCI size alignment procedure.
54///
55/// The size alignment procedure, specified in TS38.212 Section 7.3.1.0, reduces the number of different DCI sizes, to
56/// simplify the detection of DCI messages by the UE.
57///
58/// \remark CORESET 0 is assumed to be configured for the cell if \ref coreset0_bw is set to any value other than 0.

60 /// \name DCI formats and Search Spaces the UE is expected to monitor.
61 ///
62 /// Configuring the UE to monitor fallback DCI formats on Common and UE-specific Search Spaces as well as
63 /// DCI formats 0_1 and 1_1 on UE-specific Search Spaces might result in a size alignment that does not fulfill the
64 /// conditions outlined in step 3 of the DCI size alignment procedure, namely:
65 /// - The total number of different DCI sizes configured to monitor is no more than 4 for the cell.
66 /// - The total number of different DCI sizes with C-RNTI (formats 0_0, 0_1, 1_0 and 1_1) configured to monitor is no
67 /// more than 3 for the cell.
68 ///
69 /// This will trigger step 4 of the size alignment procedure, i.e., the size of the fallback DCI formats monitored in
70 /// a UE-specific Search Space will be set to the size of the fallback DCI formats monitored in a Common Search Space,
71 /// by altering the size of the frequency domain resource assignment field.
72 /// @{
73
74 /// \brief Flag for fallback DCI formats on a UE-specific Search Space.
75 ///
76 /// Set to \c true if the UE is expected to monitor fallback DCI formats on a UE-specific Search Space. If set to \c
77 /// false, the payload size for this DCI and SS configuration will not be computed, preventing DCI packing.
78 bool dci_0_0_and_1_0_ue_ss;
79 /// \brief Flag for non-fallback DCI formats on a UE-specific Search Space.
80 ///
81 /// Set to \c true if the UE is expected to monitor non-fallback DCI formats on a UE-specific Search Space. If set to
82 /// \c false, the payload size for this DCI and SS configuration will not be computed, preventing DCI packing.
83 bool dci_0_1_and_1_1_ue_ss;
84 /// @}
