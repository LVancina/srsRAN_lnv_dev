Modulates a PDSCH codeword according to TS38.211 section 7.3.1 Physical downlink shared channel.

Parameters
[out]mapperResource grid mapping interface.
[in]codewordsThe encoded codewords to modulate.
[in]configconfiguration parameters required for PDSCH modulation.
Note
The number of codewords shall be consistent with the number of layers.
The codeword length shall be consistent with the resource mapping, considering the reserved resource elements.
