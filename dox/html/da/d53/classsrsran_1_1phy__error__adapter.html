Detailed Description

Implements a generic physical layer error adapter.

Currently, the adapter only logs the error and the context.

void srsran::phy_error_adapter::on_late_resource_grid ( const resource_grid_context & context)
inlineoverridevirtual

Notifies a resource grid outside the slot window.

This error occurs when a resource grid transmission request for slot is received after the processing of slot has started.

The time window the lower physical layer can receive a resource grid for a slot starts with lower_phy_timing_notifier::on_tti_boundary() and finishes with the beginning of the processing of the first symbol within the slot.

Parameters
[in]contextContext in which the resource grid is not available.
