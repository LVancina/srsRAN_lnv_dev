80/// \brief Constructs a precoding weight matrix for one layer mapped into four transmit ports.
81///
82/// All weights are derived from TS38.214 Table 5.2.2.2.1-5, which describes one layer CSI reporting using a Type 1,
83/// single panel codebook. For one layer mapped to four antenna ports, the precoding uses the same beam for both antenna
84/// polarizations. The beam can be selected from eight possible beams, and the phase shift between cross-polarized
85/// antenna elements can be selected in \f$\pi/2\f$-rad increments, starting from 0 rad. The generated precoding weights
86/// for ports 0 and 1 correspond to the antenna elements of the first polarization, while ports 2 and 3 correspond to
87/// the antenna elements of the second polarization.
88///
89/// \param[in] beam_azimuth_id Beam selector in the horizontal plane. Corresponds to parameter \f$i_{1,1}\f$ in TS38.214
90/// Section 5.2.2.2.1. Values: {0, 1, ..., 7}
91/// \param[in] pol_shift_id Polarization phase shift selector. Corresponds to parameter \f$i_2\f$ in TS38.214
92/// Section 5.2.2.2.1. Values: {0, 1, ..., 3}
93/// \return A precoding weight matrix for one layer and four ports.
94precoding_weight_matrix make_one_layer_four_ports_type1_sp_mode1(unsigned beam_azimuth_id, unsigned pol_shift_id);

96/// \brief Constructs a precoding weight matrix for two layers mapped into four transmit ports.
97///
98/// All weights are derived from TS38.214 Table 5.2.2.2.1-6, which describes two layer CSI reporting using a Type 1,
99/// single panel codebook. For two layers mapped to four antenna ports, the precoding uses one beam for each layer.
100/// The first beam can be selected from eight possible beams, and the second beam is specified by applying an offset to
101/// the direction of the first one, in \f$\pi\f$-rad increments, starting from 0 rad. The phase shift between
102/// cross-polarized antenna elements can be selected in \f$\pi/2\f$-rad increments, starting from 0 rad. The generated
103/// precoding weights for ports 0 and 1 correspond to the antenna elements of the first polarization, while ports 2 and
104/// 3 correspond to the antenna elements of the second polarization.
105///
106/// \param[in] beam_azimuth_id First beam selector in the horizontal plane. Corresponds to parameter \f$i_{1,1}\f$ in
107/// TS38.214 Section 5.2.2.2.1. Values: {0, 1, ..., 7}
108/// \param[in] beam_offset_id Second beam offset selector. Corresponds to parameter \f$i_{1,3}\f$ in TS38.214 Section
109/// 5.2.2.2.1. Values: {0, 1}
110/// \param[in] pol_shift_id Polarization phase shift selector. Corresponds to parameter \f$i_2\f$ in TS38.214 Section
111/// Section 5.2.2.2.1. Values: {0, 1}
112/// \return A precoding weight matrix for two layers and four ports.
113precoding_weight_matrix
114make_two_layer_four_ports_type1_sp_mode1(unsigned beam_azimuth_id, unsigned beam_offset_id, unsigned pol_shift_id);
