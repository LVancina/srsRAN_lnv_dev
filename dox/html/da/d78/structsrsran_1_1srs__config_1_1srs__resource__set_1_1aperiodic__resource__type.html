uint8_t srsran::srs_config::srs_resource_set::aperiodic_resource_type::aperiodic_srs_res_trigger

The DCI "code point" upon which the UE shall transmit SRS according to this SRS resource set configuration. See TS 38.214, clause 6.1.1.2. Values {1,..,maxNrofSRS-TriggerStates-1}, max_nof_srs_trigger_states_minus_1.

optional<unsigned> srsran::srs_config::srs_resource_set::aperiodic_resource_type::slot_offset

An offset in number of slots between the triggering DCI and the actual transmission of this SRS-ResourceSet. If the field is absent the UE applies no offset (value 0). Values {1,..,32}.
