Adds the CRC to the a transport block, carries out segmentation and computes all codeblock metadata for later use (encoder and rate matching).

First, the transport block CRC is attached, as per TS38.212 Section 7.2.1. Then, the transport block is split into a number of segments and, if needed, a CRC is attached to each segment_tx. This is done according to TS38.212 Section 5.2.2. The function also computes other segment_tx metadata (e.g., coded and rate-matched length) according to TS38.212 Section 5.4.2.1.

[out]described_segmentsSegments (unpacked, one bit per entry) and corresponding metadata.
