◆ system_time_throttling

float srsran::lower_phy_configuration::system_time_throttling

System time-based throttling.

Determines a minimum baseband processor period time between downlink packets. It is expressed as a fraction of the time equivalent to the number of samples in the baseband buffer. Set to 0.9 to ensure that the downlink packets are processed with a minimum period of 90% of the buffer duration.

Set to zero to disable this feature.

◆ time_alignment_calibration

int srsran::lower_phy_configuration::time_alignment_calibration

Time alignment calibration in number of samples.

Models the reception and transmission time misalignment inherent to the RF device. This time adjustment is subtracted from the UL-to-DL processing time offset for calibrating the baseband device.

Remarks
Positive values cause a reduction of the RF transmission delay with respect to the RF reception, while negative values increase it.
