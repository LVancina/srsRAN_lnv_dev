31/// \brief Compute the PRACH frequency start as a function of the PUCCH guardbands.
32///

34/// PRACH frequency start is \c msg1-FrequencyStart.
35///
36/// \param user_params parameters passed by the user for the generation the PUCCH resource list.

41/// \brief Compute the number of PUCCH resources that are used for SR and CSI.
42///
43/// If the user has defined a larger number of resources for SR and CSI than needed, this will returns the actual
44/// estimated number of resources that will be used by the GNB, constrained by the maximum PUCCH grants per slot.
45/// \param user_params parameters passed by the user for the generation the PUCCH resource list.
46/// \param max_pucch_grants_per_slot maximum number of PUCCH grants that can be allocated per slot in the cell.
47/// \param sr_period_msec SR period in milliseconds.
48/// \param csi_period_msec CSI period in milliseconds.
