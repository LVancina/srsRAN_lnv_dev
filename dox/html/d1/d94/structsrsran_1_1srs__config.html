bool srsran::srs_config::is_tpc_accum_disabled {false}

Mapping to NR RRC: If true, the field is present in NR RRC indicating value disabled and UE applies the TPC command without accumulation (this applies to SRS when a separate closed loop is configured for SRS). If false, the field is absent in NR RRC and UE applies TPC commands via accumulation.
