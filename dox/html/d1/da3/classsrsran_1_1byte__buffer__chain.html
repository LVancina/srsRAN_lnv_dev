Detailed Description

This class represents the concatenation of several individual data packets. The class iterator type automatically traverses the concatenated data_packets in a byte by byte fashion.

bool srsran::byte_buffer_chain::prepend ( byte_buffer_slice slice)
inline

Prepends a byte_buffer_slice to the beginning of the byte_buffer_chain. This operation has O(N) complexity.

Returns
true if operation was successful, false otherwise.
269 {
270 if (slice.empty()) {
271 return true;
272 }
273 if (not valid() or full()) {
274 return false;
275 }
276 if (empty()) {
277 // In the empty case, append == prepend.
278 return append(std::move(slice));
279 }
280 // Create a new slice at the right, and shift all slices to the right by 1 position.
281 new (slices_ptr + slice_count) byte_buffer_slice(std::move(slices_ptr[slice_count - 1]));
282 for (size_t i = slice_count - 1; i > 0; --i) {
283 slices_ptr[i] = std::move(slices_ptr[i - 1]);
284 }

287 // Store slice in the first (now empty) position
