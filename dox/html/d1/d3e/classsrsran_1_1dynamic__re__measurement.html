Detailed Description

template<typename measure_type>
class srsran::dynamic_re_measurement< measure_type >

Dynamic container for RE measurements.

The amount of memory reserved is fixed and set at construction time.

Template Parameters
measure_typeThe type of the represented RE measurement (e.g., float or cf_t).
Warning
Instantiating an object of this class entails a heap memory allocation.
