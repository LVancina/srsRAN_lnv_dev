Detailed Description

PUSCH codeword buffer interface.

Interfaces the PUSCH demodulator with the UL-SCH demultiplexer. The PUSCH demodulator notifies processed continuous blocks of data using the method on_new_block and notifies the end of data using the method on_end_codeword.

Gets a view of the next block of demodulated soft bits.

This method is temporal. It allows the PUSCH demodulator to get a view of the demodulated soft bits storage and it avoids the copy.

Parameters

A read-write view of the next soft bits available for writing.
