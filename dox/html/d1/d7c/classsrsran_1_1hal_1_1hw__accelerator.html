Detailed Description

template<typename T, typename U>
class srsran::hal::hw_accelerator< T, U >

This interface provides the basic functions to enqueue and dequeue an operation to hardware accelerator.

Template Parameters
TType of the input data, and of the packed codeword in the case of encoders or soft-data in case of decoders, to the hardware accelerator.
UType of the output data to the hardware accelerator. Generic hardware accelerator interface definition.
