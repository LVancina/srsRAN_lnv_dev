Detailed Description

Describes a PDSCH modulator interface.

The PDSCH modulator shall modulate PDSCH according to TS38.211 section 7.3.1 Physical downlink shared channel. It provides two possible modulation granularities:

  • Codeword: it modulates a codeword and maps it.
  • Codeblock: it modulates a single codeblock from a codeword.
