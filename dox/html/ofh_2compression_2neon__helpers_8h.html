uint8_t srsran::ofh::neon::determine_bfp_exponent ( int16x8x3_t vec_s16x3,
unsigned data_width )
inline

Determines BFP exponent for the resource block passed in a vector of three NEON registers.

The content of the three NEON registers is represented in the table below. Here REx stands for one component of a unique resource element (16 bits long):

reg0: RE0 RE0 RE1 RE1 RE2 RE2 RE3 RE3
reg1: RE4 RE4 RE5 RE5 RE6 RE6 RE7 RE7
reg2: RE8 RE8 RE9 RE9 RE10 RE10 RE11 RE11
Parameters
[in]vec_s16x3Vector of three NEON registers storing 16bit IQ pairs of the resource block.
[in]data_widthCompressed samples bit width.
Returns
Calculated exponent value.
