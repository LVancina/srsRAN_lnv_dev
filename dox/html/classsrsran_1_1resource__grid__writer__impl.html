Parameters

[in]lSymbol index.
[in]k_initInitial subcarrier index.
[in]maskBitset denoting the subcarriers to be written (if true), starting from k_init.
[in]symbolsSymbols to be written into the resource grid.
Returns
A view to the unused entries of symbols.
Note
The number of elements of mask shall be equal to or greater than the resource grid number of subcarriers.
The number of elements of symbols shall be equal to or greater than the number of true elements in mask.

Parameters

[in]lSymbol index.
[in]k_initInitial subcarrier index.
[in]strideDistance between adjacent symbols. A stride of 1 means that the allocated REs are contiguous.

Note
