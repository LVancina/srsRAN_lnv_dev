unsigned srsran::downlink_preemption::int_configuration_per_serving_cell::position_in_dci

Starting position (in bits) of the 14 bit INT value applicable for this serving cell within the DCI payload. Must be multiples of 14 (bit). See TS 38.213, clause 11.2. Values {0..maxINT-DCI-PayloadSize-1}, where maxINT-DCI-PayloadSize = 126.
