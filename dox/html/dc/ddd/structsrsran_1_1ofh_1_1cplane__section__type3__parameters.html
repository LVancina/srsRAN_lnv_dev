uint16_t srsran::ofh::cplane_section_type3_parameters::time_offset

Time offset from the start of the slot to the start of cyclic prefix, see O-RAN.WG4.CUS, 7.5.2.12. For the PRACH Control-Plane message the value may refer to the start of cyclic prefix or the start of PRACH preamble depending on cpLength parameter value, see O-RAN.WG4.CUS, 4.4.3
