Detailed Description

Describes the PDSCH processing parameters.

Member Enumeration Documentation

◆ anonymous enum

anonymous enum

Reference point for the PDSCH DM-RS subcarrier index k, as per TS38.211 Section 7.4.1.1.2.

Set it to PRB0 when the corresponding PDCCH is associated with CORESET 0 and Type0-PDCCH common search space and is addressed to SI-RNTI.

Otherwise, set it to CRB0.

Enumerator
CRB0 

The reference point is subcarrier 0 of the common resource block 0 (CRB 0).

PRB0 

The reference point is subcarrier 0 of the physical resource block 0 of the bandwidth part (BWP). 

  Use this option when PDSCH is signalled by CORESET 0. For this case, the BWP parameters must align with
  CORESET0.
