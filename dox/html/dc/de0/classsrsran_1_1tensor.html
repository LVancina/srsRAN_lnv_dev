Detailed Description

template<unsigned NDIMS, typename Type, typename Index_type = unsigned>
class srsran::tensor< NDIMS, Type, Index_type >

Base tensor interface.

Template Parameters
NDIMSNumber of dimensions.
TypeType of data to store.
Index_typeData type used for representing dimension indexes. It is used in method get_dimension_size().
