Detailed Description

template<typename T>
class srsran::detail::ring_buffer_storage< std::vector< T >, true >

Specialization of internal ring_buffer data storage for the case a std::vector is used. For this specialization, the compiler does not know the size of the buffer at compile time, but it knows that it is a power of 2. Thus, it is able to avoid the % operator, and uses instead a bit-wise and.
