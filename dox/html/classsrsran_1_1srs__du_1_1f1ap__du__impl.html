async_task< f1_setup_response_message > f1ap_du_impl::handle_f1_setup_request ( const f1_setup_request_message & request)
overridevirtual

Initiates the F1 Setup procedure as per TS 38.473, Section 8.2.3.

Parameters
[in]requestThe F1SetupRequest message to transmit.
Returns
Returns a f1_setup_response_message struct with the success member set to 'true' in case of a successful outcome, 'false' otherwise.
Remarks
The DU transmits the F1SetupRequest as per TS 38.473 section 8.2.3 and awaits the response. If a F1SetupFailure is received the F1AP will handle the failure.
