Detailed Description

template<concurrent_queue_policy... QueuePolicies>
class srsran::priority_task_worker< QueuePolicies >

Task worker that can handle tasks with different levels of priority.

Each priority level gets associated a separate task queue with queueing policy defined by concurrent_queue_policy. The task worker will pop tasks starting from the highest priority queue and will continue to the next priority level if the current queue is empty. If there are no tasks in any of the queues, the task worker will wait for wait_interval before checking for new tasks.

Template Parameters
QueuePoliciesQueue policies for each priority level. The number of policies must match the number of supported priority levels.
