Detailed Description

Required parameters for packing a DCI format 1_0 scrambled by C-RNTI, CS-RNTI or MCS-C-RNTI.

Remarks
Defined in TS38.212 Section 7.3.1.2.1.
The case where the random access procedure is initiated by a PDCCH order is handled separately.
