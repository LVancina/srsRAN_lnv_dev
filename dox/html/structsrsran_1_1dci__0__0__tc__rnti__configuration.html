◆ hopping_offset

unsigned srsran::dci_0_0_tc_rnti_configuration::hopping_offset

Frequency hopping offset, as per TS 38.213 Section 8.3 and Table 8.3-1.

If frequency hopping is enabled via the frequency_hopping_flag, it must be set to one of the following values:

  • (0, 1), if N_ul_hop is set to 1.
  • (0, 1, 2, 3), if N_ul_hop is set to 2.

◆ N_ul_hop

unsigned srsran::dci_0_0_tc_rnti_configuration::N_ul_hop

Parameter , as per TS38.212 Section 7.3.1.1.1.

is the number of bits used in the DCI payload to pack the frequency hopping offset, represented by hopping_offset.

If frequency hopping is enabled via the frequency_hopping_flag, it must be set to:

  • 1, if is smaller than 50.
  • 2 otherwise.
