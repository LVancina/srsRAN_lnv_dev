static_vector<unsigned, pusch_constants::MAX_NOF_PUSCH_TD_RES_ALLOCS> srsran::csi_report_config::semi_persistent_report_on_pusch::report_slot_offset_list

Timing offset Y for semi persistent reporting using PUSCH. This field lists the allowed offset values. This list must have the same number of entries as the pusch-TimeDomainAllocationList in PUSCH-Config. A particular value is indicated in DCI.Values {0,...,32}.
