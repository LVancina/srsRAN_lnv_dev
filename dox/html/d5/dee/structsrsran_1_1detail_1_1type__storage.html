Detailed Description

template<typename T, size_t MinSize = 0, size_t AlignSize = 0>
struct srsran::detail::type_storage< T, MinSize, AlignSize >

Helper class to store, construct, destruct and access an object stored in a memory buffer. This class shouldn't be used outside of library code, as it is inherently unsafe.

Template Parameters
Ttype of object stored
MinSizelower bound for the embedded buffer size
AlignSizelower bound for the alignment of the embedded buffer
