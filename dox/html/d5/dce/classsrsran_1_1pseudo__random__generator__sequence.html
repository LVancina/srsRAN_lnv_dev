Detailed Description

Generates the pseudo-random sequence defined in TS38.211 Section 5.2.1.

The parallel generation of the state sequences and , as defined in TS38.211 Section 5.2.1, exploits the fact that these sequences have a memory of 31 terms, but only the four oldest terms contribute to the generation of the next one.
