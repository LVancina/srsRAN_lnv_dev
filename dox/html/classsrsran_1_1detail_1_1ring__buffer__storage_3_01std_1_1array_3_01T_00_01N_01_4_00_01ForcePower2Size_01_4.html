template<typename T, size_t N, bool ForcePower2Size>
class srsran::detail::ring_buffer_storage< std::array< T, N >, ForcePower2Size >

Specialization of internal ring_buffer data storage for the case a std::array is used. This specialization tries to leverage the fact that the buffer size is known at compile time. If N is a power of 2, the compiler should be able to convert the % operation into a bit-wise and.
