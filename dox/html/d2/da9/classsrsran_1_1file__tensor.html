Detailed Description

template<unsigned NDIMS, typename T, typename Index_type = unsigned>
class srsran::file_tensor< NDIMS, T, Index_type >

Binary file input interface for data with multiple dimensions.

Allows reading values of type T with NDIMS dimensions from a binary file.

Template Parameters
NDIMSNumber of dimensions of the data.
TData type.
Index_typeData type used for representing dimension indexes.

Reads the file.

The contents of the file are dumped to a dynamic_tensor using the dimensions set during construction. The file is closed when the operation is completed.
