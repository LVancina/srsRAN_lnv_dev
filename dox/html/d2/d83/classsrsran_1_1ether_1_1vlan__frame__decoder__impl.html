Decodes the contents in the given buffer into a VLAN Ethernet frame.

Decodes the contents of the buffer, filling the fields in the eth_params structure. Returns a new span with the VLAN Ethernet header removed.

Parameters
[in]frameFrame to be decoded.
[out]eth_paramsEthernet message parameters or undefined contents in case of error.
Returns
A span containing the payload of the VLAN Ethernet frame or an empty span in case of error.
