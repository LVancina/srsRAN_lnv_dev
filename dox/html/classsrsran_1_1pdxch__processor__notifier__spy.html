void srsran::pdxch_processor_notifier_spy::on_pdxch_request_late ( const resource_grid_context & context)
inlineoverridevirtual

Notifies the unavailability of a downlink resource grid.

This error happens in a sector when the resource grid for the processing slot is not available at the time when a symbol is modulated.

The time window the lower physical layer can receive a resource grid for a slot starts with lower_phy_timing_notifier::on_tti_boundary() and finishes with the beginning of the processing of the first symbol within the slot.

Parameters
[in]contextContext in which the resource grid is not available.
