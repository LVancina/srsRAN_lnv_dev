Computes the channel estimate.

Estimates the channel to the current receive antenna port for all transmit layers.

Parameters
[out]estimateThe estimated channel coefficients for all resource elements, all OFDM symbols, all receive ports and all transmit layers.
[in]gridThe frequency–time resource grid.
[in]portReceive antenna port the estimation refers to.
[in]pilotsDM-RS symbols (a.k.a. pilots). For each layer, symbols are listed by RE first and then by OFDM symbol.
[in]patternPattern with the positions of the DM-RS symbols.
[in]cfgEstimator configuration.
