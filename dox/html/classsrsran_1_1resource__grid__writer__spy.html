Detailed Description

Describes a resource grid writer spy for testing classes that write in the resource grid.

Asserts that the mapped resource elements match with a list of expected entries.

This method asserts that mapped resource elements using the put() methods match a list of expected entries without considering any writing order.

Parameters
[in]expected_entriesProvides a list of golden symbols to assert.
Note
The test is terminated in case of mismatch.

Asserts that the mapped resource elements match with a list of expected entries.

This method asserts that mapped resource elements using the put() methods match a list of expected entries without considering any writing order, while using a parametrizable maximkum error threshold.

The test is terminated in case of mismatch.

Returns

Note
