Detailed Description

IQ decompressor selector implementation.

The selector will select the IQ decompressor between the registered ones to decompress IQ samples, based on the given compression parameters.

Decompress received compressed PRBs.

Decompresses compressed PRBs from the input buffer according to received compression parameters and puts the results into an array of floating point IQ samples.
