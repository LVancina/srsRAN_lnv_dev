Detailed Description

PRACH processor main interface.

Provides a single access point to all PRACH-related interfaces, that is prach_processor_notifier, prach_processor_request_handler and prach_processor_baseband.

Note
All implementations of this interface must trigger an assertion error if the PRACH processor is not connected to a notifier before calling get_request_handler() or get_baseband().
