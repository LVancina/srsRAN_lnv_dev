Detailed Description

template<concurrent_queue_policy QueuePolicy = concurrent_queue_policy::locking_mpsc, concurrent_queue_wait_policy WaitPolicy = concurrent_queue_wait_policy::condition_variable>
class srsran::general_task_worker< QueuePolicy, WaitPolicy >

Single thread worker with a locking MPSC input task queue. This worker type is ideal for the cases where there is low contention between task producers.
