optional<uint8_t> srsran::sib2_info::abs_thres_ss_blocks_consolidation_rsrp

Threshold for consolidation of L1 measurements per RS index. If the abs_thres_ss_blocks_consolidation_rsrp, abs_thres_ss_blocks_consolidation_rsrq and abs_thres_ss_blocks_consolidation_sinr fields are absent, the UE uses the measurement quantity as specified in TS 38.304.

◆ cell_reselection_priority

uint8_t srsran::sib2_info::cell_reselection_priority

Defines the absolute priority of the concerned carrier frequency, as used by the cell reselection procedure. Value 0 means: lowest priority. Values {0,...,7}.

◆ cell_reselection_sub_priority

optional<float> srsran::sib2_info::cell_reselection_sub_priority

Indicates a fractional value to be added to the value of cell_reselection_priority to obtain the absolute priority of the concerned carrier frequency for E-UTRA and NR.Values {0.2, 0.4, 0.6, 0.8}.
