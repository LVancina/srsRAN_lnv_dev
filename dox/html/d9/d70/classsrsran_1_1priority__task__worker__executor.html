Detailed Description

template<concurrent_queue_policy QueuePolicy>
class srsran::priority_task_worker_executor< QueuePolicy >

Task executor with Priority (lower is higher) for priority_task_worker.

This executor does not depend on the priorty_task_worker<QueuePolicies...> to avoid increases in template instantiations being proportional to all possible QueuePolicies... permutations.

Template Parameters
QueuePolicyEnqueueing policy for the given priority level.
