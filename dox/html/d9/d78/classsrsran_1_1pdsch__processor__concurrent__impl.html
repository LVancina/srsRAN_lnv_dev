Detailed Description

Implements a PDSCH processor with concurrent codeblock processing.

Remarks
The number of PDSCH codeblock processor instances contained in cb_processor_pool must be equal to or greater than the number of consumers in executor. Otherwise, an assertion is triggered at runtime.
