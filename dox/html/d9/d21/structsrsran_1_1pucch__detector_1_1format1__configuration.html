◆ initial_cyclic_shift

unsigned srsran::pucch_detector::format1_configuration::initial_cyclic_shift

Cyclic shift initial index {0, ..., 11}.

Index used to retrieve the cyclic shift for generating the low-PAPR sequence. Specifically, it corresponds to parameter in the formula for the cyclic shift in TS38.211 Section 6.3.2.2.2.

◆ n_id

unsigned srsran::pucch_detector::format1_configuration::n_id

Pseudorandom generator initialization seed {0, ..., 1023}.

Corresponds to parameter in TS38.211 Section 6.3.2.2.1.

It must be set to the higher layer parameter hoppingID (see TS38.331 Section 6.3.2 — Information Element PUCCH-ConfigCommon) if it is configured. Otherwise, it must be equal to the physical cell identifier .

◆ nof_harq_ack

unsigned srsran::pucch_detector::format1_configuration::nof_harq_ack

Number of expected HARQ-ACK bits {0, 1, 2}.

This parameter should be set to zero when trying to detect a positive scheduling request only.

optional<unsigned> srsran::pucch_detector::format1_configuration::second_hop_prb

PRB index used for the PUCCH transmission within the BWP after frequency hopping {0, ..., 274}.

Index of the PRB posterior to frequency hopping as per TS38.213 Section 9.2.1, if intra-slot frequency hopping is enabled, empty otherwise.
