Detailed Description

Keeps a pool of dynamic_format_arg_store objects. The main reason for this class is that the arg store objects are implemented with std::vectors, so we want to avoid allocating memory each time we create a new object. Instead, reserve memory for each vector during initialization and recycle the objects.
