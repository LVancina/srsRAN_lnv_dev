void re_pattern_list::merge ( const re_pattern & pattern)

Merges a given resource element pattern into the list.

This method tries to combine a new pattern with the existing ones in the list according to if

  • the resource block allocation parameters are identical, and
  • the symbol mask or the resource element mask is identical. Otherwise, the pattern is appended in the list.
