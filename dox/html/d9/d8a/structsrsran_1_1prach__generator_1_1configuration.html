unsigned srsran::prach_generator::configuration::root_sequence_index

Root sequence index.

Parameter to select the PRACH sequence number from:

  • TS38.211 Table 6.3.3.1-3 with range {0, ..., 837} for long preambles, and
  • TS38.211 Table 6.3.3.1-4 with range {0, ..., 137} for short preambles.
