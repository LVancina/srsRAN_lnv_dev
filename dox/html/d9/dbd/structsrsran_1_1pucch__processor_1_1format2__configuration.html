◆ n_id_0

unsigned srsran::pucch_processor::format2_configuration::n_id_0

DM-RS scrambling identity {0, ..., 65535}.

Corresponds to parameter in TS38.211 Section 6.4.1.3.2.1.

It must be set to the higher layer parameter scramblingID0 given by TS38.331 Section 6.3.2, Information Element DMRS-UplinkConfig, if it is configured. Otherwise, it must be equal to the physical cell identifier .
