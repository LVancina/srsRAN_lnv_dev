DCI formats and Search Spaces the UE is expected to monitor.

Configuring the UE to monitor fallback DCI formats on Common and UE-specific Search Spaces as well as DCI formats 0_1 and 1_1 on UE-specific Search Spaces might result in a size alignment that does not fulfill the conditions outlined in step 3 of the DCI size alignment procedure, namely:

  • The total number of different DCI sizes configured to monitor is no more than 4 for the cell.
  • The total number of different DCI sizes with C-RNTI (formats 0_0, 0_1, 1_0 and 1_1) configured to monitor is no more than 3 for the cell.

This will trigger step 4 of the size alignment procedure, i.e., the size of the fallback DCI formats monitored in a UE-specific Search Space will be set to the size of the fallback DCI formats monitored in a Common Search Space, by altering the size of the frequency domain resource assignment field.

Detailed Description

DCI configuration parameters required to perform the DCI size alignment procedure.

The size alignment procedure, specified in TS38.212 Section 7.3.1.0, reduces the number of different DCI sizes, to simplify the detection of DCI messages by the UE.

Remarks
CORESET 0 is assumed to be configured for the cell if coreset0_bw is set to any value other than 0.
