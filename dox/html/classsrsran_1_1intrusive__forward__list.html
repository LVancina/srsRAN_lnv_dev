Detailed Description

template<typename T, typename Tag = default_intrusive_tag>
class srsran::intrusive_forward_list< T, Tag >

Forward linked list of pointers of type "T" that doesn't rely on allocations. It leverages each node's internal pointer (thus intrusive) to store the next node of the list. It supports push_front/pop_front, iteration, clear, etc.

Template Parameters
Tnode type. It must be a subclass of intrusive_forward_list_element<Tag>
Taguseful to differentiate multiple intrusive lists in the same node
