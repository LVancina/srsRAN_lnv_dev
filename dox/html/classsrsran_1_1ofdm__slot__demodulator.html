Detailed Description

Describes an OFDM demodulator with slot granularity.

Remarks
Performs OFDM demodulation as per TS 38.211 section 5.3.1 OFDM baseband signal generation for all channels except PRACH.
In addition to demodulation, it applies phase compensation as per TS 38.211 section 5.4 Modulation and upconversion.
