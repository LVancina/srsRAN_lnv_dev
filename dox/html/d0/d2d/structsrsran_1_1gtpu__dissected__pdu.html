Detailed Description

Intermediate representation of a received GTP-U PDU to access the unpacked header and to the raw content of the extension headers. This container holds a buffer with the original PDU to which the views of the extension header content are mapped. After processing the extension headers, the T-PDU can be extracted via gtpu_extract_t_pdu which advances the PDU buffer to the start of the T-PDU according to hdr_len.
