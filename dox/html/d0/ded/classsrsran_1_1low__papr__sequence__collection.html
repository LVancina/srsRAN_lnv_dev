Detailed Description

Describes a low PAPR sequence collection as per TS38.211 Section 5.2.2.

This class describes a collection of sequences. The collection of sequences is generated at construction time. The constructor is not suitable for real-time operation while the sequence retrieval through method get() is suitable for real-time execution.
