◆ nof_cb_preambles_per_ssb

uint8_t srsran::prach_appconfig::nof_cb_preambles_per_ssb = 4

Indicates the number of Contention Based preambles per SSB (L1 parameter 'CB-preambles-per-SSB'). See TS 38.331, ssb-perRACH-OccasionAndCB-PreamblesPerSSB.

Remarks
Values of cb_preambles_per_ssb depends on value of ssb_per_ro.

◆ nof_ssb_per_ro

float srsran::prach_appconfig::nof_ssb_per_ro = 1

Indicates the number of SSBs per RACH occasion (L1 parameter 'SSB-per-rach-occasion'). See TS 38.331, ssb-perRACH-OccasionAndCB-PreamblesPerSSB. Values {1/8, 1/4, 1/2, 1, 2, 4, 8, 16}. Value 1/8 corresponds to one SSB associated with 8 RACH occasions and so on so forth.

◆ prach_frequency_start

optional<unsigned> srsran::prach_appconfig::prach_frequency_start

Offset of lowest PRACH transmission occasion in frequency domain respective to PRB 0. To minimize interference with the PUCCH, the user should leave some guardband between the PUCCH CRBs and the PRACH PRBs. Possible values: {0,...,MAX_NOF_PRB - 1}.

◆ preamble_trans_max

uint8_t srsran::prach_appconfig::preamble_trans_max = 7

Max number of RA preamble transmissions performed before declaring a failure. Values {3, 4, 5, 6, 7, 8, 10, 20, 50, 100, 200}.
