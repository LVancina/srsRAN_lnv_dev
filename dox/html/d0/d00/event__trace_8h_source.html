32/// The event tracing framework allows to trace any kind of event inside an
33/// application.
34/// To enable event tracing the ENABLE_SRSLOG_EVENT_TRACE macro symbol should
35/// be defined, otherwise calls to the tracing framework will be ignored. This
36/// is important to avoid the overhead of tracing when it is not required.
37/// For details about each event trace type please refer to:
38/// https://docs.google.com/document/d/1CvAClvFfyA5R-PhYUmn5OOQtYMH4h6I0nSsKchNAySU/edit
39
40/// Initializes the event trace framework saving the trace events to a
41/// "event_trace.log" file.

43
44/// Initializes the event trace framework using the specified log channel to log
45/// all trace events.
